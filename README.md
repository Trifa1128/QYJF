# 南瞻部洲（1）：天圆地方

2020/09/17
如果说《七印解封》是从时间维度对历史进行回应，那么我们今天开始连载的《南瞻部洲》将从空间角度对当下进行回应。过去人类历史留下很多关于宇宙和我们人类所处空间的传说，例如四大部洲、封神演义、山海经等，很多都是真的，但也有很多走了样，很多已经失传。我们从今天开始连载的《南瞻部洲》，将为大家展现人类生存的部分真实环境，同时在南瞻部洲这个大环境下为大家揭示今天人类社会发生的各种大事背后的部分真相。

中国古代神话故事说“盘古开天劈地”，这是真实的故事。人类生活的那块地是在盘古开天劈地造出来的那些地的位置上，但它并不属于盘古开天劈地的那些天和地，而是在创世主造就的三界内的最低一层的那块地上。先有了盘古开天劈地，然后经过漫长的岁月，宇宙发生了巨变，然后在盘古造就的天地中出现了创世主创造三界这么一件生命、宇宙、一切的一切有史以来从未发生过的事情。创世主创造的三界奥妙无穷，亘古以来从未有过，开天劈地头一回，普天众神用尽他们的智慧也难以参透其中玄机。中国神传文化中说的九重天、三十三层天都是这个三界内的事情，而极乐世界、须弥山以及西方的天国等等它们都属于类似于盘古开天辟地那样的体系。

“玉皇大帝”他是三界内最低一层的主，我们《南瞻部洲》系列就围绕这一层天地来说。关于这一层天地其实中国已经有非常多的故事，而且说的都很详细，只是人们不相信而已。对于这一层天地，中国神传文化中说它是“天圆地方”，后人不理解，就认为这是古人因为看到我们头上蓝天象圆的，地一望无际象是方的所以说“天圆地方”；也有种观点认为“天圆地方”是根据中国阴阳学说而来的一种形象比喻。

《周髀算经》说“方属地、圆属天，天圆地方”，《晋书・天文志》说“天圆如张盖，地方如棋盘”。这些说法很有道理。其实不仅仅中国有地是方的说法，在西方也有这样说的，约翰《启示录》在打开第六印时这样描述，“此后我看见四位天使站在地的四角，执掌地上四方的风，叫风不吹在地上，海上，和树上”，这句话就已经非常明白地说了地有四角有四方，地是方的。其实，这层天地它的外貌真的是“天圆地方”这个样子的，是古代修炼人通过天目、或者他直接元神离体、或者他肉体飞升看到的三界内最低一层天地的真实模样。

“地方如棋盘”，确实这个样子。三界内最低一层的那块地是方形的，中间就象被人在上面纵横划了两条线分出四个方块，人类居住所在的那个方块被称为南海，南海右边是西海，西海右边是北海，北海右边是东海。在每个海那里都有一个象陆地板块那样的地，如果我们把整个南半球看成四海的其中一个海的话，那么这个陆地板块的大小就类似于澳洲那么大。南海里的那个板块叫南瞻部洲，东海里的那个板块叫东胜神州，北海里的那个板块叫北俱芦洲，西海里的那个板块叫西来贺州。整个地球是被埋在南瞻部洲那个板块里面，就象一块小石头。当然，四海那里也有一些小岛，这些岛如果和四大部洲比起来就象地球上那些突出海面的珊瑚礁。所以整个地一眼望去，看到的就是四个方形的大海中有四大部洲。

我们在《七印解封》曾经说过，四大部洲原来都是在一个水平上的，后来北俱芦洲升上来了，而再后来，南瞻部洲就掉下去了。所以如果我们站在南瞻部洲这块地的表面，就会看到一个壮观的景象，那就是这块地的周围那些南海的水象巨大的瀑布往下流。

构成四大部洲的那个土也并非我们地球上看到的那个土。这个土的成分非常复杂，也是奥妙无穷。我们肉眼看到的这个地球上土、岩石等等，属于南瞻部洲那个土的一种，而我们肉眼看到的那个什么都没有的空间、我们抬头看见的“天”，其实也是它那个土，只是这种土对于人的肉眼来说就象是透明的。它那里还有很多不同的土，里面有由这些不同的土构成的不同生命生活在其中，所以一眼看去，人类地球这个位置上的土中同时同地和人在一起生活的生命那真是太多了。

居住在南瞻部洲里的生命不是象我们人类肉眼看到的我们人是站在地球上，其实这些生命都是被埋在南瞻部洲里，包括人类。而那些能够在南瞻部洲上面站着的，或者在上面空中飞着的，对于埋在南瞻部洲里的生命来说他们就是神，神通广大。虽然被埋在土里，但我们人类以及那些被埋在里面的生命他们并没有觉得自己被埋在土里，我们人觉得自己是站在地上的，然后头上那个人类看不见的土构成的天空，里面也有非常舒服的天气，也有闪电雷鸣，也是奥妙无穷。南瞻部洲里的生命制造出来的各种飞行器，或者人类制造的汽车等等，如果从南瞻部洲外面看，那么就会看到它们其实都是在土里钻，在土里爬。

对于被埋在土里的生命来说南瞻部洲是无穷无尽的大、无穷无尽的厚。其中生命，无论是往哪里去，是往上还是往下，往左还是往右，其广阔是这里的生命穷尽他们多少生都走不完，再快的宇宙飞船都飞不到尽头。但如果是修炼人，那就不受这个限制，他不仅可以出南瞻部洲，修得好甚至还可以跳出三界得正果。

在四海中间，也就是用来划分四海的那个纵横分界线的交汇处，那里是一个象巨大的火山口那样的存在，海水进不来，里面深不见底。这里是这一层次的魔王居住之地。这里的魔王并非人们常说的地狱或者冥府之类的，这里的魔王他是为维护这一层天地而存在的。

虽说四海组成的地是方的，但它的功能就象圆的，假如有人能够到达东海的尽头，那么他会发现他以为的尽头却是西海那一头，然后他从西海那头一直走，他就能走回他出发之地。但四海太大了，作为里面的神来说都很难飞遍，而且在海的尽头，也就是四海组成的这块地的边缘都是火，这些火不是这里的生命能够逾越的。

玉皇大帝就是这块四海组成的地上面的神，他上面有广阔的天，他可以在这个天地里飞。这个天地之大，无论玉皇大帝怎么飞，他也飞不出这个天，就象西游记里的孙悟空，他翻了无数的筋斗云，他以为他已经飞到天边了，其实还在这个天地里。跳出这个天地看，就会看到玉皇大帝所在的天是圆的，这四海组成的地就在这个圆的中间，整个这个圆的表面看起来象是被厚厚的白云包住，玉皇大帝等神仙就生活在这个圆里面，用尽他们的神通也出不去，这里面也是奥妙无穷，生机勃勃。

# 南瞻部洲（2）：中国王兵分四路

2020/09/20
如果说最低一层的南瞻部洲上面是正面，那么它的背面对应的就是中国人说的“阴间”。地球以及地上人类是被埋在南瞻部洲里面的土中，而在南瞻部洲这块地的上面是一个比地球更加丰富多彩的世界。这个世界是一个真实的世界，那里的生命每天也有他们丰富多彩的生活，那里的历史也是荡气回肠。这个世界的存在直接影响着埋在土里的那个地球上的万事万物的存在，中国神话传说中的很多事情都和南瞻部洲上面这个世界直接相关，我们《七印解封》说的中国王他也是在这个南瞻部洲上，那条为祸人间的大红龙也在南瞻部洲上，那个半山腰有个禁闭大红龙小屋的那个山，它也在南瞻部洲上。南瞻部洲上的这些万事万物和地球上的人类连在一起，它们之间的联系有些象这么一个样：当拘禁大红龙的那座山倒了，天安门那块地就陷下去了。

南瞻部洲上的中国城非常大，当我们这里说“城”的时候，大家还是把它理解为国家会比较好些，我们用“城”这个词是对应着约翰《启示录》来说的，它其实就是象人类的国家一样。从中国城这一头飞到另一头，对于那个空间的生命来说，比如中国王骑着他坐下的金麒麟飞过去，大概也要好几个月。就象地球，森林里有野兽，海中有鲨鱼，南瞻部洲上面也是很险恶的，上面的山中水里也经常有各种妖魔鬼怪。

为了便于大家理解，我们《南瞻部洲》系列如非特别说明，后面所有涉及的地名都统一以对应到人间的那个位置的名字来说，这也是当初《封神演义》的写法。

我们在《七印解封（12）：中国王复国》说过中国王在创世主弟子的帮助下已经开始了复国之路，我们看到，当初中国王从台湾出发是兵分四路。其中第一路由中国王自己带领，从台湾渡海来到福建，然后往左沿着海岸线，从越南入云南的时候，中国王自己就率领一只精兵从那里深入中国内部，从云贵那里一路往北京方向去，目前已经到了贵州境内。而同时这一路的大部队由一位长着六只眼的将军带领，继续沿着中国城的界线经青海、西藏到内蒙古。这位六眼将军骑银麒麟，金盔金甲，金盔上有一个由红色火组成的缨，手拿一把大刀。这一路在他们沿途经过之处，在一些重要的关隘都会留下一支部队在那里等待命令，当他们接到进攻命令后就会攻打那个关隘进入中国。现在这些部队很多都已经在攻打那些关隘了。



第二路领兵将军身穿银盔银甲，座下银麒麟手拿两柄三尖叉。他这两柄三尖叉是个能带着这位将军和他坐下麒麟一起飞的宝贝。他们这一路将士坐着很多战船渡海从吉林珲春那个地方登陆，然后沿中国界限绕黑龙江到内蒙古与第一路的将士会合。他们沿途经过之处，在一些重要的关隘也都象第一路一样留下一支攻打关隘的部队。当第一路和第二路将士来到蒙古会合时，一位蒙古族将军穿着蒙古族武士衣服，坐下一匹木色的战马，手拿一把大砍刀，带着一队蒙古神箭手已经等候多时。

第三路领兵的将军和第一路那位六眼将军几乎一模一样，象孪生的兄弟，不同的只是这位将军戴的那个头盔是两个蓝色火构成的缨。他们这一路是和中国王那一路一起坐着很多战船到福建那个地方登录，然后他们就与中国王分开，往右沿着海岸线一直往朝鲜那个方向走，经浙江、江苏、山东到天津，在沿途经过的重要关隘他们也都象第一路那样留下一支攻打关隘的部队。目前这位六眼将军已经带着大部队从天津那里攻进去了，可以预计天津不久就会有大事发生。

这三路的每一路都有上万的将军，他们模样也是各种模样，但基本上都是人样。他们座下全是麒麟，各种麒麟都有，而手上武器也是各种各样。这三路所有跟着将军的士兵脚下都穿着一双特别的鞋子，有了这双鞋子，他们的行军速度就非常快。

第四路由六艘非常大的船组成，他们是中国王派往世界各国的使团，其中每艘船上带队的是中国古代书生的模样，但没有戴书生帽，腰上都有一把宝剑，而跟随的将士背上都有一把弓，每艘船上都放着六座宝塔，外形象托塔李天王手上那个玲珑宝塔。这一路六艘船从台湾一起出海，来到一个地方后就分开奔赴他们各自出使的国家。

# 南瞻部洲（3）：诛仙无敌阵

2020/09/22
中国王复国之事牵扯甚广，与《封神演义》周室当兴有几分相像，但根本上却相差甚远。

读过《封神演义》的都知道，商朝末年纣王无道周室当兴，于是有昆仑山弟子姜子牙出山辅佐明主改朝换代，看似这件事情只是人间朝代兴衰，但如果从整个南瞻部洲来看就会发现，周朝当兴却又是南瞻部洲修炼人的杀劫。于是，姜子牙带领人间周朝士兵是否能够克敌制胜，从被埋在地里的人来说，是由谋士运筹、将士英勇、士兵血战换来的，但《封神演义》却告诉了人们这么一个道理，埋在土中的周朝军队和商朝军队的胜负是由南瞻部洲地上面的阐教与截教这两家修炼人的神通法力的比试来决定的。于是，维护旧时代的截教摆下各种阵法，作为推动新时代的阐教修炼人就必须去破阵，这是他们修炼要过的关必须要面对没有选择。所以当截教门人摆下十绝阵，阐教众门人就纷至沓来。随着阵的立，埋在土里的周朝将士就会损兵折将攻城失败，而随着阐教门人成功阵破，埋在土里的周朝士兵就能克敌拔城。因为周朝当兴是历史的必然，所以截教立下的阵时辰一到必然就有高人出来破它，但过程中却是对阐教修炼人的真实考验，就如九曲黄河阵一般，燃灯道人、云中子过了此劫，其他十二门人都被削去三花。就这样，当我们把空间放大到整个南瞻部洲来看的时候，修炼人封神与周室当兴看似八辈子打不到一起的两件事就成了一件事。

周室当兴是天意的必然，但过程中有多少劫难要出现、要死多少人却又是由修炼人的心决定。修炼人的心就象这些劫难的开关，如果修得好，人间劫难就少，截教的那些阵就摆不了，修炼人本身要过的劫难也就少，反之就是人间生灵涂炭、截教的阵一个连着一个、修炼人在劫难逃。就象《封神演义》说的，元始天尊吩咐姜子牙回去路上如果有人叫他千万别回头，否则就会招来三十六路征伐。故事我们都知道了，姜子牙没能守住而回了头。这一回头，就象一把钥匙，把那些早已等候的三十六路征伐的门打开了，不仅人间生灵涂炭枉死了很多人，而且还直接导致了自己十二位同门被削去三花，还有很多门中弟子以及其它门派弟子失去生命。姜子牙这一回头从人的角度来说好像没什么，但从修炼角度来说却并非回头还是不回头这件事本身，而在于姜子牙作为修炼人犯了最严重的错误：不遵师命。

《封神演义》的主角是谁？是武王伐纣还是修炼人？当初《封神演义》里的修炼人是在南瞻部洲的地上面，周室当兴是在土里面，而今天中国王复国之事正好相反：中国王复国是在南瞻部洲的地上面，修炼人被埋在土里面！

《封神演义》截教共设下“十绝阵”、“九曲黄河阵”、"诛仙阵"、“万仙阵”这四大阵，但今天身在中国的修炼人所要面对的大阵何止四个，多到修炼人都数不过来。在中国那个地方，大阵套小阵，阵又连着阵，而设下这些大大小小阵的那些阵主，层次都非常的高，很多都是三界之外很高层次上来的，而且很多这些阵主不是在另外空间，它们被保护着就和人一样直接就在土里，但它们的神通是开放的。大大小小的这些阵密密麻麻连成相互交叉的立体的网，把整个中国网在其中。所有这些阵合并一起，我们从七印书卷看到，这个阵有一个恶名，叫“诛仙无敌阵”。这个阵之大，是历史从未有过的，估计未来也不可能再有了。那个安排“诛仙无敌阵”的邪恶它是什么来头、是佛是道是神还是魔，我们从七印书卷看不到，只看到它千变万化不知道哪个是真正的它。

南瞻部洲地面上的中国王要复国，看似是由中国王的他那些将士们奋勇杀敌决定的，或者就象人们以为的，在人类空间是由美国总统、由欧洲领导人、以及全球正义力量组成反共联盟来实现，其实那些都只代表美国、欧洲他们自己的选择，但他们并非是那个决定性的因素，也不会因为真组成了一个反共联盟中国就会改朝换代，就象我们知道的，朝鲜、古巴、委内瑞拉、伊朗等等，哪一个都被封锁的够严了，经济上也都够糟糕透了，但它们现在都还没倒，为什么？把人类放大到整个南瞻部洲就会看到，不仅中国，整个世界现在实际上都很象当初《封神演义》演的那样，那个真正决定中国、决定当今人类社会变局的那个决定性力量，是由地上那些修炼人与那些设下劫难的邪恶的正邪较量决定的。

# 南瞻部洲（4）：谁在统治中国？

2020/09/23
创世主创造的三界奥妙无穷，宇宙中无论多高层次的神也无法完全洞悉，包括当初那些制定“七印书卷”的众神，所以在他们制定“七印书卷”这部历史剧本的时候，也是反反复复的安排人类历史进行测试，然后不断修改，然后一直来到了本次人类文明时期，也就是一切都要结束了，终局要上演了，宇宙众神还是没能完全洞悉三界的奥妙，他们制定的第七印到今天还索性都被封上了。

埋在南瞻部洲土里的人虽然是宇宙最低层次的，但人体也是奥妙无穷，三界内不同层次的空间都和这个人体有着密切的联系，相互作用。我们上回讲的中国王复国的那个空间，也就是南瞻部洲这块地的地面上，那个世界里有一个中国，我们把上面的中国叫中国城，原因之一是与约翰《启示录》中说的那座被大淫妇占据的城对应上，还有一个原因是因为在那个空间中国这片土地是有一座象城墙一样的界线把中国和外面区分开来的，所以外形上也确实是一个巨大的城，城里有山有河有城。如非特别说明，我们《南瞻部洲》以及《七印解封》所说的中国城就指这个在南瞻部洲地面上的那个中国。

中国城里有很多生命，生活在地球上的中国人，在那个空间中国城里一般都有一个生命与地上这个人有个对应，但中国城里的这个人的身份、外形、干的事情和地球上那个人就不一定一样，例如现在地球上很多人看着是人，在那个空间看与他对应的那个生命可能不是人，不少可能是当初跟着大红龙从天上被打下来的大红龙的那些使者部众。那个世界的生命很多长的象人样，也有很多长的不是人样，那个空间就是这样的。那个空间很象古代中国，古代中国的房子，古代中国人穿的衣服，等等。这个中国城里的人和地球上这个人的肉体关系非常直接，或者我们可以这么说，人这边的肉体很多时候是被南瞻部洲地上面那些生命控制的，如果控制了那边的生命，地球上的这个人差不多就算是被控制了，除非地球上的这个人他的主元神很清醒，或者他来源的那个地方的神很强大足以影响到人间，或者他自己能够分辨出来哪个是他真正自己的思想，等等，但这对于一般人来说太难了。

在人间我们知道中国似乎是被中国共产党控制了，而作为共产党的党魁，相当于整个中国都被共产党党魁控制了一样，那么实际情况是怎么样的呢？

在中国祁连山靠近青海湖那里一座荒无人迹的山脚下有个山洞，洞里有个修道人，它用功能把洞口封起来了，一般人看不见，而那些能看到这个洞口的人如果未经同意擅自闯入，那么洞口顶上就会发出一个象闪电一样的能量把那个闯入者的头盖击穿。这个道平时穿着黑色的道袍，头上带着紫色的道冠，上面有一个发簪，这个发簪是属于这个道的一个比较厉害的法器。这个道已经修炼了有上亿年，相当于造就本次地球的时候它就来到南瞻部洲修炼了。它这个洞里面不是很大，洞中间有一个红白蓝黑黄五色毯子，它平时就坐在上面面对洞口打坐修炼。在它的右手边的洞壁上张贴着一副画，上面是一个红黑太极图，左手边洞壁上也挂着一幅画，上面画着一位道，象三国演义时谋士打扮，估计是它的师父。我们目前从七印书卷看不到它这个师父的来头，可能以后会看到，如果到时候看到了我们再说它的来源。

我们从七印书卷看到，这个道它始终处于整个“诛仙无敌阵”的保护之下，似乎这个道它是整个“诛仙无敌阵”中非常关键的部分，牵扯到整个“诛仙无敌阵”的方方面面。

我们在《七印解封（3）：拘禁在天安门的大红龙》说过，当大红龙被拘禁在北京天安门对应到另外空间的那座山中小房子的时候，这条大红龙把自己身体分成两条，一条就在天安门所对应的另外空间这个小房子里等待时机，另一条来到西方。当共产主义来到中国后，两条大红龙就合并为一条，中国王与大红龙就在南瞻部洲这个地面上展开了守城与攻城的恶战，最后中国王被迫逃到台湾那个地方，而大红龙就和它的部众占据了中国城，这条大红龙就坐上了中国城的王位宝座。中国人从那时候开始，不仅在人间，中国在南瞻部洲存在的层层空间也都被大红龙统治了，在人间表现形式就是思想上中国人被共产主义控制，社会上那些共产党员身体对应在南瞻部洲地面上中国城里的那个生命，很多就是大红龙的那些使者和部众，它们在中国城里属于统治阶层。如果在中国城里把那些大红龙使者和它的部众全部灭了，那么人间的共产主义、共产党也就要消失。

现在中国城里那个王位宝座上，之前是中国王的宫殿里，后来就被那条从死亡世界打下来的大红龙占据了，而现在坐在上面的不是当今中共党魁对应到那个空间的身体，也是一条大红龙，但这条大红龙并非从天上死亡世界被打下来那条，它是假的，是那个在祁连山修炼了上亿年的那个道在南瞻部洲这个地面上按照那条真的大红龙的模样新造出来的。这个真相连南瞻部洲地面上绝大部分生命，包括还在那里控制地球上人类的大红龙的使者和它们的部众都不知道，今天我们把这个真相写出来，估计那个世界的生命也就会很快知道。

那么那条真的大红龙、被第二印打开时那个骑红马的邪恶在死亡世界里造出来的那条大红龙去了哪里？

在这个道居住的洞里，当这位道念动咒语就会有一个门出现在洞中，然后这位道走进这个门，它就会立即出现在中国的另一头的一座山的山洞里，这座山在中国叫庐山，这个山洞在庐山的半山腰。这个山洞是圆形的，对着洞口的另一头的地上有三个方形蒲团叠在一起，然后上面又有很多个圆形的蒲团叠在上面，这个道就坐在这个叠起来的蒲团上打坐。而在这个道的头顶，那条真的大红龙被很多钉子钉在这个洞的顶上，大红龙的头就对着这个道的脸。而在大红龙左边和右边还有两个生命也被钉在洞顶山，大红龙的身后又有三个生命被钉在洞顶山。然后在这个道打坐的左手边的洞壁上，又有三个生命立着排成一排被钉在洞壁上，右手边也这样被钉着三个生命，其中右手边被钉的最靠近这个道的那个就是当今中共党魁在南瞻部洲中国城对应的那个生命。这个洞共钉了十二个生命，所有被钉的样子都是耶稣当初被钉十字架那个样。具体另外十个被钉的生命是谁我们后面找个时间再细说。

就这样，那个道就在这个洞里，通过控制被钉的这十二个生命控制着南瞻部洲那个中国城里的那条假的大红龙和它的众多使者统治着中国城，以及地球上的中国。中共党魁在中国被封为“一尊”，其实也蛮可怜！

# 南瞻部洲（5）：宝塔神兵

2020/09/27
南瞻部洲那个地上面有一个很大的海，这个海不是南海，按照地球的说法在大陆板块内的应该叫湖，但这个湖实在太大了，我们还是把它叫做海比较合适。《山海经》有个海内经、海外经、大荒山经，那么这个海内经所说的海估计就是指这里了，具体我们目前还没细看，当我们《七印解封》写到《山海经》的时候我们再详细的核实一下。这个海对应到我们这个空间就是地球。在这个海里有十九块很大的陆地，就象地球上的大陆板块一样，其中一块地在海的中央，另外十八块地星罗棋布地分布在中间这块地的周围。整个地球所有的国家都在这十九块地上，而中国城在中央那块地上面，所以如果从南瞻部洲看就知道地球上的中国叫“中国”真是很对的，它就是在这个海的中央那块地上。在地球上中国人要去美国很远，去日本很近，但在那个地方去日本和去美国的距离差不多，美国就在台湾的后面，并没有象在地球一样隔着一个太平洋。
