# 七印解封（21）：失败带来教训

2020/09/13
有些圣经故事说耶稣为全人类受苦赎了全人类的罪，我们从七印书卷没看到这种事情。耶稣救了约5万人，其中57%约3万人属于犹太人，按照当初耶和华与犹太人债主的约他们很多债是被免除不会加到耶稣身上，而在剩下的约2万名非犹太人中，也有很多债主是和耶稣善解的，所以实际上耶稣要承担的是在2万人以内。这个数量的罪已经达到了耶稣法力的极限，于是耶稣在传道三年后就必须要离开人间，否则他自己也要被毁在其中。

有些圣经故事说耶和华无所不能，我们从七印书卷也没看到这种事情。耶和华没有超越西天宝座上那位时而佛时而魔的天意，在犹太人对其它民族犯下的罪面前，耶和华也没有能力免除，在西方社会的这些民族国家中，阿拉伯对应的体系不是耶和华能够掌控的，他们被安排来和耶和华体系在西方做对立，至于作为地上与西方并行存在的东方，属于道这个体系的中国就和耶和华更遥远了。

我们从七印书卷看到，当初耶和华为犹太人定的约中，他要达到的目标是耶稣作为犹太人的弥赛亚赎下犹太人整个民族的罪，然后人与神共通的时代就会来到犹太人的国家，神话般的世界将出现在以色列，以色列也因此会成为西方社会的表率，引领着整个西方社会一直等到末后大红龙制造的那个让普天下受试炼的巨难到来的时候，在创世主以“人子”现身拯救世人之时。为了这个目标，耶和华安排了摩西引领犹太人出埃及，留下十诫希望犹太人保持和神的联系，留下神迹与弥赛亚的预言希望犹太人能够在耶稣来的时候能够认得耶稣就是他们等待的弥赛亚从而接纳他而被救。耶和华用尽他的智慧和法力，呕心沥血苦心经营着这一切，但我们今天已经知道，因为那些阻挡人类得救的邪恶旧势力它们全力破坏，耶和华的计划失败了。

因为旧势力的破坏，摩西带领的犹太人历史文化被毁灭；毁灭历史文化这不是最严重的，因为即使文化毁灭，如果耶稣能够象释迦牟尼一样在以色列地区传法49年，那么失去历史的犹太人在最后大部分也可能会相信耶稣就是他们的弥撒亚，耶和华的计划也就可以实现，但因为旧势力的破坏，以色列地区居住了很多非犹太人，由这些非犹太人带来的罪直接导致了耶稣不得不提前结束传道匆匆离开人间。当然这里只是列了两个主要原因，还有正神没能在外星人问题上获胜、西天宝座换了神等等其它的原因。

当时犹太人虽然大部分没能认耶稣为弥赛亚，但这个结果不能完全算在他们身上，而且耶稣和犹太人弥赛亚这个历史不是真正的得救，它是来预言历史的，来为末后全世界面临的、由大红龙制造的那个让“普天下人受试炼”的那个巨关巨难的时候为“人子”的到来、为真正得救预备道路的，所以犹太人没认耶稣也不是什么要命的问题，这不会成为他们的罪，犹太人还可以好好地活下去，以色列这个国家还能够存在。这段历史它只是犹太人和耶稣一起配合去实现耶和华计划的这个事情上，不是犹太人不够努力，不是耶稣没有尽力，大家都尽力了但没能实现。真正导致犹太人犯下新罪的是他们同意让耶稣钉十字架。罗马不同意，耶和华警告了，但犹太人还是要这样干，那就是他们的选择，那就是他们自己犯下的罪。因为这个罪，犹太人从此在人类社会经历了一段段比出埃及更加悲惨的赎罪历史，直到二战被纳粹残酷虐杀那都不是偶然的，都是因为同意把耶稣钉十字架直接导致的后果。而那片让耶稣钉十字架的地，耶路撒冷这个地方，也因此同罪，要在炮火反反复复轰炸下赎罪。

虽然耶和华计划失败了，但这个失败却恰恰给今天人类提供了一个非常好的历史教训：今天的基督教、今天的全世界各大宗教的教徒，今天的人类，是不是也正在重走当初犹太人对待弥赛亚所走过的路？

等待两千年，当大红龙制造的那个让“普天下人受试炼”的巨关巨难真的来了，当救世主以“人子”的身份真的来了，犹太人当初没认耶稣还有机会，但此时如果不认“人子”可能就永远没机会了。

# 七印解封（22）：约翰《启示录》

2020/09/14
圣经故事说耶稣死后又复活，我们从七印书卷看到这是真的。但复活的肉体已不是当初那个肉体了，是一个纯净的肉体。耶稣以复活肉体的方式在门徒中行了神迹之后就回到天上，升天的时候直接把肉体化为无形。

在耶稣飞升之前，天上那场米迦勒与大红龙之战也结束了，给人得救制造巨难的大红龙及其使者也终于到达人间，西方也因此即将在外星人与大红龙的双重逼迫下，从悬崖那里掉下深渊，西方正神已经没有能力阻挡后面要发生的事情，也只能看着西方社会正快速地被魔掌控。于是，此时天上衆神就把目光都轉向了东方，因为在东方，保护外星人的魔还不能来到这里，大红龙也不能来到表面，神傳文化在这个地方正在陸續上演，于是那里就成了现在众神的期盼的希望了。

当初跟着耶稣从天上下来的使者有很多，他们排成两排跟在耶稣身后走出天门。这些下世的使者中，有些是安排来做弟子的，也有安排来做敌人的，反正好坏都有。施洗者约翰在耶稣右边使者中排第十，使徒约翰在耶稣左手使者中排第一。使徒约翰晚年被流放到拨摩岛，七印书卷看到，他被关在拨摩岛某个岩石下面的一个洞建造成的一间牢房，那间牢房的地是用很多平整的石板铺成的，在其中一块石板下面压着一本书。现在那本书还在那个石板下面，书皮还是好的，只是书里面爬满了虫子。如果有人能找到那个洞，那么他应该可以在那个石板下面找到那本书。约翰一直被关在这间牢房不准出来，那里有看守，对约翰并不是很好，到吃饭的时间就过来扔些水和食物给他。

我们从七印书卷看到，在一个晚上，约翰听到有声音叫他，然后他就看到天上打开了一个洞。我们从七印书卷看到，这个洞是创世主打开的，因为书上就直接写着打开这个洞的那位神的名，那个名就是创世主在末后拯救世人时，作为“人的儿子”的那个名。这个名具体是什么我们留待后面时机成熟时再揭示。约翰看到直通天庭的洞，就立即被创世主用神力把他的元神提到了一位神的宝座前。就象约翰在《启示录》所描述的，“我立刻被圣灵感动，见有一个宝座安置在天上，又有一位坐在宝坐上。看那坐着的，好像碧玉和红宝石。又有虹围着宝座，好像绿宝石。宝座的周围，又有二十四个座位，其上坐着二十四位长老，身穿白衣，头上戴着金冠冕。”我们从七印书卷确实看到是这个样子，但《启示录》说“又有七盏火灯在宝座前点着，这七灯就是神的七灵”就不对了，是有“七盏火灯”，但那不是神的七灵，这应该是约翰自己想的。

约翰在《启示录》没有描写这位坐宝座的长什么样。我们从七印书卷看到，那位坐在宝座上的是一位女神，属于道家，她就是中国神话故事里的女娲，坐在宝座上的她头上戴着的是十二星的冠冕。我们在前面讲过的那位生下中国王的妇人，她就是这位女娲神派到地上的使者。而现在约翰来到的这个地方，它其实不是天上某个世界里的神殿，而是天上一座非常玄妙的山中的一座神仙洞府里。当时这座神仙洞府里确实坐了二十四位长老，但他们不是属于女娲洞府里的神，他们是从不同世界来的，其中女娲右手边坐着的是道家体系的，左手边是佛家体系的，但他们全部给约翰显示的是西方神的形象，而创世主在这个洞中给约翰显示的是穿着当时约翰熟悉的西方人的衣服的样子。为什么叫“羔羊”？因为此时创世主头部显示的形象是“有七角七眼”的，那个角很象小小的羔羊角，如果只是侧面看，就象一只羔羊角下长着一只眼睛，约翰应该是从这个角度来看的，因为他也不知道这位神具体是谁，所以他就说“羔羊”。

《启示录》说“我看见坐宝座的右手中有书卷”，我们从七印书卷看到那本“七印书卷”并非在女娲手上，而是在女娲右手边坐在第二位的那位道手中。而且这部“七印书卷”只能由这位道拿着，其他神不可以，因为当初众神制定这部七印书卷时，这位道是被大家指定出来保护这部书卷的。虽然这部书卷由这位道拿着，但他也打不开。

既然大红龙来到了人间，那么也就到了这部书卷可以解密的时候了。约翰是当时地上唯一来这里看这部书打开的，而且是创世主特别让他上来的，为什么他会有这个荣耀具体我们也不是很清楚。

当那位道拿出这部七印封严的书卷，就有天神出来向全天宇众神发出谁能打开的信息，所以《启示录》就写“我又看见一位大力的天使，大声宣传说，有谁配展开那书卷，揭开那七印呢”。于是全天宇都知道女娲洞府里正在邀请有能力的神前来打开这部七印书卷，但是“在天上，地上，地底下，没有能展开能观看那书卷的。”见普天下没有能打开的，于是创世主就上前亲自打开了七印书卷，并把该约翰知道的内容一一展现给约翰，让约翰把大红龙毁灭众生与创世主拯救世人的信息传达到地上，于是人类也就知道了大红龙、世界末日、万王之王万主之主、审判等等知识了。

天上的七印书卷打开后，当时天上有些神就被允许根据打开的七印书卷制作了一些副本。故事写到这里，我们今天也才知道原来我们看的那本也是副本，正本还在那位道手上。

# 七印解封（23）：七印下的安排

2020/09/15
约翰《启示录》有很多内容是从四活物或者二十四位长老或者其他天使说出来的话，也有不少内容是约翰根据自己的理解写出来的意思，那些内容不在七印书卷之内，只代表他们各自的认识，所以我们也就不说了，我们就只针对七印打开的情况先简单的交代一下。

第一印打开，“见有一匹白马，骑在马上的拿着弓。并有冠冕赐给他。他便出来，胜了又要胜。”约翰看到的这个映像是七印书卷给他浓缩出来的概括。我们从七印书卷看到，这个封印之下的历史是关于开创三界的。这位骑白马拿着弓戴皇冠的因为创造了三界，所以后来就又被众神称为创世主。创世主在开创三界时，每开创一层就有很多生命出来阻拦，于是创世主就带着很多神在层层开创中层层打仗，一边打仗一边造三界，最后都是创世主胜。至于开创三界的恢弘历史，我们这里就掠过，将来如果有机会再展开说。

第二印打开，“就另有一匹马出来，是红的。有权柄给了那骑马的，可以从地上夺去太平，使人彼此相杀。又有一把大刀赐给他。”创世主开创三界有创世主的目地，当创世主开创了三界后就有一位恶神被安排来搞破坏，这位骑红马的就是那位被安排来破坏的。第二印下的历史就是有关它怎么造大红龙、怎么安排大红龙和大红龙使者层层下来直到人间为祸众生的历史，这些历史我们七印解封有些已经说到了，后面还会继续说。这位恶神因为是直接与创世主为敌的，所以在前几年一场创世主弟子与它的大战中已经被处理了，其中它体系的绝大部分被投入无生之门，它的那把大刀现在还在天上，具体大战的历史我们留到后面写终局有关最后的正邪大战的时候再说。七印书卷里写的“圣徒”指的是那些属于创世主的弟子。

第三印打开，“我就观看，见有一匹黑马。骑在马上的手里拿着天平。”这位骑黑马拿天平的神负责保持三界内各方的平衡。从七印书卷看到它骑的其实更象黑色的骡子，而且这位神它就象喝醉酒了一样，坐在骡子上也左摇右摆，说话也象喝醉了那样含含糊糊。它在修炼人能看到的最高层次也是这个样：骑着黑色骡子的醉汉。第三印下的历史就是关于它怎么平衡三界各方势力、怎么维护阴阳平衡相生相克的。因为它的所为干扰了创世主要做的，现在也被处理了。

第四印打开，“我就观看，见有一匹灰色马。骑在马上的，名字叫作死。阴府也随着他。有权柄赐给他们，可以用刀剑，饥荒，瘟疫，〔瘟疫或作死亡〕野兽，杀害地上四分之一的人。”这位我们大家应该都能猜到了，它是一位魔王。它是根据三界内的生命所为而施行报应的，所以叫“死”。第四印下的历史就是关于它怎么根据生命各自的所为而施行报应的历史，包括战争、天灾人祸、富贵贫穷、生老病死等等。因为它的所为同样干扰了创世主要做的，现在也被处理了。

第五印打开，“有为神的道，并为作见证，被杀之人的灵魂。”这一印的历史是关于创世主带着他的弟子在人间怎么与骑红马制造出的大红龙一伙进行争战的，具体我们留到写终局的时候再展开说。

第六印打开，“此后我看见四位天使站在地的四角，执掌地上四方的风，叫风不吹在地上，海上，和树上。我又看见另有一位天使，从日出之地上来，拿着永生神的印。他就向那得着权柄能伤害地和海的四位天使，大声喊着说，地与海并树木，你们不可伤害，等我们印了我们神众仆人的额。”这一印的历史是关于创世主弟子在末后救度众生的，具体我们也留到写终局的时候再展开说。

第七印在约翰的时候是被创世主打开的，但现在又被关上了，而且不仅我们看到的这部副本是关上的，修炼人还特意跑到拿正本那位道那里核实了一下，他那本正本第七印也是同样被关上的，所以我们认为可能是创世主把当初约翰看到的第七印内容作废了，未来如何将由现在的人类从新选择，所以我们也就不说了。

# 七印解封（24）：创世主临凡

2020/09/19
我们在《七印解封（4）》说过，当那位大红龙想要吞吃的男孩坐上中国那片土地对应到另外空间那座城的王位宝座后，对应到人类空间，历史也恰好来到了黄帝时期。

昆仑山在中国神传文化中被称为天下第一神山、万山之祖，这不是比喻，更没有高估。《山海经·海内西经》说昆仑山有开明兽守之，《搜神记》说昆仑之墟外被弱水围绕，《穆天子传》说穆天子见西王母于昆仑丘，这些都是真的。中国那座昆仑山对应到靠近人类这一层空间，它就是这样的，外面弱水环绕，门口有开明兽，那个开明兽和山海经里描写的一模一样，象老虎，但老虎头上有九颗人脸的头，挺吓人，西王母就在开明兽守着的那座宫殿里。但是昆仑山被称为第一神山并非因为西王母住在那里，而是因为这座山奥妙无穷，众妙之门，横跨三界内每一个层次，穿透三界直到三界之外。

在三界之外，昆仑山之顶是一个火湖，湖中央有一座只能容几个人的小岛，岛上站着创世主，一会儿显示为手拿银枪身穿银盔银甲的神将，一会儿又显示为法相庄严的佛，一会而又显示为各种各样形象的神，一会儿这个样一会而那个样，非常快地变化着，天上众神无法看到创世主的真正模样。

火湖外有一个入口，入口那里立着一个半人高的、绿玉打造的方形柱子，柱子四面都写着很多字，上面放着一个被封印的书卷。从入口到火湖中的小岛有一座桥相连，这座桥是由一块块象木板那样的东西悬在火湖上组成的。火湖中的火是特别造就的，没有创世主的特别许可，无论层次多高的神都不可能越过火湖去到那个小岛上。

火湖外面，数不清的神越过重重天险来到这里等着，什么样的神都有，什么世界来的都有，多遥远体系来的都有，绝高层次的神都有，但他们都被拦在火湖外。也有不少不相信的神不听劝阻想直接越过火湖上那座岛，但所有做这个尝试的在火湖上就立即被化为齑粉。而在那座桥入口旁边也站着很多神，他们是想从那座桥过到那座岛上但都在入口那里被拦下了，因为入口那里有一个无形的盾牌一样的门，要想过那座桥，就必须从这个门进去。这个门的钥匙就是绿玉柱上那个书卷，只要打开书卷那个门就会开，但那个书卷上的封印是创世主封印的，只有那些右肩膀上受了创世主印记的神才能打开。

有不少神穿过遥远的宇宙匆匆来到火湖边，他右肩膀上在久远历史之前就早已带有创世主授的印记，他就直接来到入口那里，打开绿玉柱上的书卷，然后上桥去到岛上，那些火湖外等待的神无不羡慕之极。也有在那些等在外面的数不过来的神当中，有一位神他之前并没有被创世主授印记，可这时候他突然发现自己右肩膀上出现了一个创世主授的印记，他“欣喜若狂”，这个词不太好但我们也就只好先用这个词来形容了，这种欣喜是宇宙中任何东西都无法换取的。他就立即飞到入口那里打开书卷上桥来到小岛上，那些等待的众神看到他被授印记来到岛上，那是久久都没能抹去的无比羡慕的心情了，或者用人的语言来比喻就是“久久难以平静”。

岛上有块平整的石头，石头上有个四方形的宝贝，上面有个入口，入口可大可小，就刚好比来到这个宝贝面前的那位神的身体大一点。当那位被授记的神来到岛上，他向创世主行礼后就从这个入口一跃而下，然后他就发现自己已经站在一块广阔的、白云一样的、悬在空中的地上，上面已经有许许多多从那个入口进来的神站立在那里了，而地下面是一望无际的海。

能来到这里的神，都是蒙创世主之恩被授记选上的，他们将从这里跟随创世主下世到中国那个地方开创辉煌的神传文化，为世人在末后大患难中得救开路。

这是一个无比壮观的画面。

白云地的四个角上，是众仙女在载歌载舞，那个美妙只应天上有。而蒙恩被选的众神上空，一层层的飞天在上面盘旋飞舞，象一圈圈五颜六色彩虹，那些天上各种美妙的花就从彩虹那里飞下来，光彩夺目美丽非凡，

蒙恩被选的众神后面，是西方天国众天使组成的乐队，各种西方乐器应有尽有，前面，是东方天国众天使组成的乐队，手上各式东方乐器在那里演奏，东西方合璧奏出的天籁层层天宇也难有。

蒙恩被选的众神左面和右面是列队欢送的众神，各样盛装服饰应有尽有。每一边的欢送众神都分成前后两组，手腕上戴着各种响器，举到头上摇摆发出各种美妙声音。当左右两面两组欢送众神列队相对而行时，左面就有凤凰带着各种各样五颜六色的神鸟，盘旋飞舞忽左忽右，从他们中间穿过飞向蒙恩被选的众神，而同时在右面就有龙和麒麟带着天上各种神兽盘旋忽上忽下飞向蒙恩被选的众神，然后神鸟神兽在蒙恩被选众神头顶上交替盘旋飞舞。当左右两面两组欢送众神背对而行时，凤凰就带着众神鸟盘旋飞出外面，而龙与麒麟也带着众神兽盘旋飞出外面，就这样交替进行，那个景象人间从未有过的好看。

海上还有很多星罗棋布的小岛，岛上面又坐了很多在三界之内不同空间修炼的修炼人，他们都在那里盘腿打坐。

同时整个白云地、整个海其实都是在不断往人间下走的，但白云地上蒙恩被选的众神却完全感觉不到这种变化。而随着白云地的下走，还有不断被创世主授记的神从那个小岛入口那里跳进来，但他们一跳进来，无论整个白云地下走到那里，他们都会立刻来到这块地上。当整个这个白云地下走到三界最低层次，众神即将要转生人间之时，创世主就从火湖岛那个入口跳了下来，而那个入口就随即关闭，被创世主授记同下人间开创中国神传文化的机缘就这样永远关上了，那些外面众神无比惋惜，用人话来说捶胸顿足了。那边入口一关上，这边创世主身穿黄色的中国皇帝衣，头戴皇帝冠冕出现在白云地上，然后带着蒙恩被选的众神飞下白云地转生人间。

# 七印解封（25）：大洪水灭世后

2020/09/24
在世界各地的不同民族间均有关于大洪水的传说，埃及人、希腊人、中国人、波利尼西亚人、爱斯基摩人、格陵兰人、非洲人、印度人、美洲印第安人，等等，而最为著名的就是《圣经》故事中的诺亚方舟了。我们从七印书卷看到，这场毁灭上一次人类的大洪水发生在距今约5000年前，那时候人类道德在整体上败坏而被神以大洪水的方式毁灭。

上一次人类文明也分东西方，就象我们之前说过的，西方对应着天上佛这个体系，东方对应着天上道这个体系。上一个文明时期中国那个地方生活的是黄种人，就象中国神话传说那样由属于道家体系的女娲造的，然后经过漫长的岁月流传下来。在上一个文明时期，东方社会的生活方式很象古代中国人那样，即使是历史来到了洪水灭世前，那时候的人也还是住在象中国古代那样的建筑里，不同的是那时候的房子比本次文明时期中国古代人盖的要高很多，有很多层。那时候的社会没有电，也没有汽车、电脑之类的东西，士兵使用的武器象古代人一样都是刀枪剑戟之类的，也有比中国古代发达的强弓硬弩，还有一些非常厉害的发射类武器，大街上走的是马车，屋里有各种非常好看精致的照明灯具，各种用具非常丰富，是和现代人依赖现代化工具完全不同的生活方式。那时候的东方社会道家文化非常发达，所用文字和本次人类文明中国人的象形文字一脉相承，八卦、易经等等几乎人人皆知，也有很多习武之人，各种不同门派的真功夫，不象现代人练的那些花拳绣腿。

东方社会在那场大洪水中也没能幸免，绝大部分人都死了，中国那个地方只有青海、西藏、新疆一带高山上住的人、以及在大洪水来的时候能够逃上这些高山的人才幸运地活了下来。面对这个突然而来的大洪水，那些能够幸运活下来的人不相信神也要相信了。当然，本身这些能够留下来的人道德也还是属于没有完全败坏的，他们也才有那个机缘被神留下来。那时候地球的气候分布和现在也不一样，在青海、西藏、新疆并不是寒冷的地方，山上没有雪都是森林，新疆那里也没有沙漠，都是肥沃的土地，森林遍地，河流在其中蜿蜒流淌。而且在大洪水来之前，新疆、青海等地上非常繁荣，到处都是城市。大洪水来了之后，所有这些都被毁掉了，高山上幸存的人就依个人所长互相护持互相帮助，有擅长武艺的就负责保护大家的安全，有会种地的就在山上开垦种地，有会建造的就在山上建造房子，等等，他们在山上这样苦等了约一年的时间，然后大洪水才从青海、西藏、新疆这些地方褪去，而往下还是汪洋一片。在大洪水来之前，新疆是这一带最繁华的地区，所以当新疆的地面从洪水露出之后，山上的绝大部分人就从山上来到新疆这个地方，但之前人类所造的一切都被洪水毁灭干净，于是他们就开垦造房，开启了本次人类文明的历史。

司马迁写《史记》的第一章就叫作《五帝本纪》，第一个记载的人就是轩辕黄帝，这是对的。然后在唐代的时候有一个叫司马贞的人，他写《三皇本纪》，认为在本次人类文明黄帝之前还有三皇。这个三皇说法很容易让人以为本次人类文明开始，人类是从什么都不懂的野人那样发展起来的，房子也不会盖、火也不会生、什么植物也不认识，更没有文字。按司马贞的说法把伏羲、神农和女娲作为三皇。伏羲造八卦造字，神农尝百草，女娲造人补天。也有不同的说法认为这个三皇中应该有有窠氏，他教人盖房子，还有燧人氏教人用火。其实这些人都有，但都不是本次人类文明时期的。本次人类文明时期中国人一开始就已经有了文字，也会盖很好看的房子，生火那都是小事一件，也不需要什么尝百草，那些大洪水留下来的人脑袋里早就装满了关于各种植物动物之类的各种知识，周易八卦之类是他们的基本知识。所以他们去到新疆那个地方之后，很快就有了城市的规模，然后随着洪水的消退，而且也可能是因为大洪水的影响，新疆开始沙漠化，而新疆、青藏等地的气候也开始越来越冷，于是就有很多人往中原各个地方迁移，慢慢的在中国各地出现了大大小小数不过来的小国家。

然后当历史来到距今约4700年前的时候，在中国那里还发生了一件人神共通的大事。
