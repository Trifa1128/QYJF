# 南瞻部洲（6）：扫除外星人之战

2020/09/29
我们知道南极上面都是冰川没人住，为什么中国皇会派使者专门送去一座宝塔？原来南极那个巨大的冰盖下面有一个南极国，生活着小人。他们和我们人类是在同一个空间，也是我们人的肉身，不同的只是身高。他们被封闭在冰层底下，如果人类能够把南极那个冰盖揭开，那么我们就可以过去和他们握手拥抱了。这个南极国对应到南瞻部洲中国皇他们那个空间也有一个南极城，那里有一个南极王，他那里也有很多大红龙的使者和部众，他也需要复国，所以中国皇就派了使者给他送来了宝塔神兵。

中国皇他们是在南瞻部洲的地上面，他们对人类影响极大，我们认为可以说他们是和被埋在土里的人类生死相连的。除了他们，对人类影响极大的还有外星人。外星人和人类一样被埋在土里，它们看不到中国皇他们的存在，更看不到神的存在，它们也看不到人体的全部结构，只能看到一点点，但就这一点点人体的真相就让它们垂涎欲滴。它们也看不到地球的奥妙，更无法洞察地球和三界不同层次的那种联系，它们只看到地球在它们那个空间体现出来的那一点，但就这一点点，已经让它们觉得地球就是全宇宙最美丽的仙境了。因为它们星球如果用人眼去看，那就和垃圾堆差不多。它们也不是人类生存的这个三界物质产生的，它们是从过去盘古那样的宇宙体系在最低层象垃圾一样的物质产生出来的生命。因为旧势力的邪恶把这些外星人放进人类生存的这个范围，它们看到人体的完美看到地球的美好，就想占有人体占有地球。人类有句话叫“癞蛤蟆想吃天鹅肉”，用这句话都是抬举了它们。

说到这里我们就不得不特别提一下旧势力，它们真是非常的恶毒，为了给世人得救制造巨难，专门安排了第二印去给人从天上制造了大红龙一伙，然后又在第四印安排了魔王保护从宇宙最底层的垃圾里产生出来的外星人进入人类生存的范围控制人类。来自宇宙高层的大红龙与来自宇宙最底层的外星人上下夹击，给人类得救制造的巨难，这是三界创世以来从未有过，开天辟地头一回！

但好在现在和人类社会相关的一切正在发生巨变。相对于大红龙一伙，外星人的解决还是比较容易的，因为只要把保护外星人的魔清理了，然后再把来到地球的外星人清理了就解决了，而大红龙一伙牵扯的面太广太深，解决起来就复杂很多。我们在七印解封说过，那个负责执行第四印的那个魔已经被处理了，也就是说在背后保护外星人的那些力量已经被解除了，剩下的就是怎么清除那些来到地球表面的那些外星人。

这些来到表面的外星人它也看不到那些保护它们的魔被处理，它们不会自己离开地球，因为地球上的人类都已经被互联网、被电子科技统治了，地球和人类已经是到手的香饽饽，它们无论如何都不会自己放手。我们看到，当初外星人进地球的通道是在太阳那里，在太阳那个星球表面有条通道直通外星人它们那个世界。外星人在太阳那里建立了一个巨大的基地，这个基地就象是外星人与它们世界的中转站，然后通过这个中转站，它们在月亮那里建立了一个基地，然后在地球海洋底下也建立了一个基地。有人说地球上那个百慕大三角可能是它们的基地，其实那个地方和外星人没有关系，但那个地方和上一次大洪水灭世有关系，那里曾经是放四大部洲南海的水进地球来的其中一个口，后来负责这个口的神“无意”犯了一个错，而导致这个口经常出现一些神秘现象。在这些外星人基地那里都有庞大的机器，所有地球上的电脑、ipad、手机等产品，我们看到，电脑背后有五条线，ipad背后有四条线，手机背后有三条线，反正所有这些智能设备都有线连到月球基地上或者连到海底那个基地上，然后这两个基地又有线连到太阳那个基地上。这些线外面有一层材料包着，是外星人它们那个空间的材料，里面就象电流一样，所有人类的电子设备都被这些线控制着。如果把人类存在的物质称为一个八度的话，那么外星人这些线是比人高一个八度，如果人类有仪器能够探测高八度的物质存在，那么就能够探测到整个人类社会到处都是外星人布下的这些线，比城市里密密麻麻的电线还多。

但人的眼睛又看不到外星人，外星人却又和人一样被埋在同一块土里，神不可能转生到外星人那个空间把外星人清理了，人也不可能去到那个空间把外星人赶走。怎么办？我们看到大约是在今年第一季度的时候，从人类社会里的创世主弟子的修炼中产生出来一种开天辟地从未有过的生命，数量之多都数不过来。这种生命属于外星人所存在的那个空间，但他们和外星人又完全不一样。我们知道，人类的生命外形是神所特有的，在过去，外星人所在的这个空间是不允许有象人类这样的外形的生命的，但现在从创世主弟子的修炼中产生出来的这种生命，他们都是人的外形，和人长的一样，我们认为他们将是未来主宰这一层的生命，是外星人这一层生命的神。我们看到，这些生命有王，他是全身赤裸的，身体就象人的肉体，脚上穿着一双鞋子，头上没有头发是光的。除了王长这个样，其他所有的生命身体都长着一种象黄铜片一样的甲，头上也没有头发，但光着脚。这些生命突然出现在地球上，在外星人那个空间，他们手掌能够发出象激光一样的能量，全面展开了扫除外星人的战斗。我们看到外星人就用它们的激光武器攻击这些生命，但那些激光根本打不穿他们黄铜一样的甲片。到今天我们看了一下，原来太阳、月亮、海中的基地上都是外星人，现在都已经是这些生命在控制了，而战斗还在继续，但已经不在太阳系里了，这些生命跟着那些逃跑的外星人从太阳那个通道追击到外星人所存在的世界中去了。

可以说，本次人类社会的现代科技就到此为止了，不会再有什么突破。这些从创世主弟子修炼产生出来的生命目前也没有毁坏太阳、月亮、海中原来外星人基地的设备，而是让它们继续运作着，等待着人类的选择，等待着最后的结束。

# 南瞻部洲（7）：美国选举之战

2020/10/05
在南瞻部洲地球对应的那个海上有一个很小的孤岛，他就是中国皇当初逃避大红龙追杀的避难之地，对应到人间就是台湾。在那个孤岛的后面又有一大片陆地，它就是美国和加拿大所在之地。美国在那个空间有一位王，他坐在他的王位宝座上治理着属于美国的这座城，各方面都按部就班地发展，然后历史来到了2020年的美国大选。

人这里选举是大家投票，但对应到南瞻部洲地上面就不同了，那里就是一场大战一般。具体来说就是由美国王的文武大臣推举两位候选人，然后由美国城里的将士们自行选择他们愿意跟随哪一位，选好后这两位候选人就带着各自的将士在一个指定的战场上拉开阵势对战，谁获胜那么对应到人间的那位就会当总统。我们看到，这场对战的结果是其中一方是以大比分获胜的。按照以往历史，当这个结果出来后，失败的一方就会欣然接受，然后人间就会按照这个结果演化。也就是说，美国选举这件事情，根本上就是美国王他自己国内的事务，谁当总统，谁不是总统，都是美国王内部决定的事情，也没有说当总统的一方一定就比不当总统那一方好，就都是他内部的事情，总统是好是坏一般是由他整个王国的道德状态决定的，或者由历史发展到那一步需要一位好领导人来扭转乾坤或者需要一位坏领导人来败坏。但今年这场选举却出了意外！

地球上的每一个国家，只要地上有创世主的修炼弟子，我们看到这些修炼人的功会形成一个保护罩把这个国家对应的层层空间包起来，于是这个国家的邪恶就出不去，外面的邪恶也进不来，相当于把这个国家的邪恶和外面隔绝了一样，而且保护罩会发出法光清除那些企图进出的邪恶。但如果这些国家的修炼人修的有漏，那么这些保护罩就会有漏洞，漏越多，保护罩的威力就会大打折扣，各种各样的邪恶就会进进出出。我们看到全世界这些保护罩其实都有漏，不同的只是漏的多和少、强与弱。导致漏洞的原因也是综错复杂，很多时候保护罩上出现的洞是土里修炼人多种执著心共同作用下的结果。另外，我们看到保护罩上的漏洞数目和位置也总是在变，比如我们一会儿看到整个中国就某个地方上空保护罩那里有一个漏，一会儿中国很多地方都会出现很多漏洞，美国和其它国家那些保护罩也差不多这样。而三界之内的大红龙势力，大部分都已经被中国那里的修炼人包在中国对应的层层空间里了，对应到南瞻部洲就是中国城里的大红龙势力都出不去中国城，中国城外的邪恶势力也进不来。

我们看到，三界之外有一位道，他穿白色道服，有金色拂尘，有一把宝剑，一直闭目坐在他的洞府中注视着人间的一举一动，此时这位道并不属于给人类得救制造麻烦的一方，也就是他并不坏，应该算是好的，他只是在那里看。当美国选举对应到南瞻部洲地面上的那一方失败后，这个道看到了中国那里的保护罩和美国那边的保护罩有些漏洞里包含了美国选举因素在里面，于是他就被这些漏洞触动了，换句话说他就变坏了。具体这些漏洞怎么触动他让他变坏我们就不去细查了，反正被触动变坏了，然后它就离开自己的洞府来到俄罗斯上空，坐在一块白云上打坐一动不动。在它的不动之下，我们看到它直接操控了南瞻部洲美国城里失败的那一方，于是，南瞻部洲失败的一方就断然拒绝接受失败的结果，公然挑战美国王。美国王大怒，下令将士处置违背王命的一方，此时，那个在中国庐山操纵大红龙的那个亿年老道就从中国城里调了一支死亡部队从中国那些保护罩的漏洞那里穿越出来，然后又穿越美国保护罩那些漏洞去到美国城帮助失败的一方去攻打美国王他们。所以，这场总统选举本来只是美国王内部事务，就这样演变成美国王与大红龙死亡部队之战了。

这只从中国城里来到美国的死亡部队我们看了一下，全部都是由死亡生命组成，它们的身体就象死了好多年没有腐烂的尸体，死皮死肉包着骨头，没有任何表情，看着非常恐怖。那个领兵的死亡之将穿黑色盔甲，头盔上有一只死亡之鹰，手拿银色枪腰上一把金色剑，骑着一匹死亡之马。

在很高层次上有一位道，具体多高层次我们也一时查不出来，它的法就象紫色的火一样穿透层层空间来到人间，变成了里面的生命生存的一个环境一般，所以人类也好象泡在它的这些紫色火组成的海洋里。当那只死亡部队离开中国城来到美国的时候，这个制造紫色火海的道就在美国制造了很多象武汉肺炎那样的病毒，以配合美国整个选举形势的变化，同时又给修炼人要做的事情制造难度。因为人间就象泡在它的紫色火海里一样，所以它想在人间给人制造出什么样的病症就只是它的一念。这个道能够干成这件事也是因为人间有大量修炼人出现了一个比较大的漏。

南瞻部洲上美国王那里的战斗打的很激烈，对应到地球上的美国也同样激烈，天象表现就是美国那些天灾人祸一个连一个没有间断，选举情势一波三折。一开始美国王将士们被大红龙的死亡部队打得节节败退，但当蓬佩奥在尼克松图书馆发表了演说后，美国地上那些人背后对应的不同来源的体系就加入到美国王一边，于是美国王带领的将士就开始反攻大红龙的死亡部队，但后来又出现了变化，更多的死亡部队穿透中国和美国那些保护罩的漏洞来到了美国，此时，中国皇使者带来的宝塔神兵也送到了美国王手中，从宝塔里出来神兵的领兵将军身穿金盔金甲坐下白色神马，腰上配一把神剑手提亮银枪，他们来帮助美国王反攻，继而十月二日那个制造紫色火的道让特朗普突然出现了武汉肺炎症状。

战斗就这样步步升级。

# 南瞻部洲（8）：人类环境的险恶

2020/10/08
三界最低层次的地是四方形的，被划分为东海、南海、西海、北海四个海，天是圆的，玉皇大帝那一层次的神仙生活在这个范围内。在南海之内有个南瞻部洲，南瞻部洲之内又有一个海，海中有十九块地，这个海就是对应到人类所在的地球，地球上的所有国家都对应到这十九块地上。约翰《启示录》描写第六印打开时说，“此后我看见四位天使站在地的四角，执掌地上四方的风，叫风不吹在地上，海上，和树上。我又看见另有一位天使，从日出之地上来，拿着永生神的印。他就向那得着权柄能伤害地和海的四位天使，大声喊着说，地与海并树木，你们不可伤害，等我们印了我们神众仆人的额。”对于天圆地方的三界最低一层来说，这四海也属于地。约翰《启示录》描写的这四角就是东海、南海、西海、北海那四个角，这件事情早已经发生。“地与海并树木，你们不可伤害”这里说的“地与海并树木”，指整个天圆地方的这块地，包括组成四海的海、组成四大部洲的土、包括上面的所有生命以及地球及人类都要被毁灭。“日出之地”指东方，创世主在东方临凡，永生神的印指创世主的法旨。

从创世主叫停毁灭四海至今已经过去了很长时间，那四位天使现在怎样了？我们看到，这四位天使就象被定住一样一直在那四个角上一动不动。其中南海之角的那位天使是站立着单手立掌于胸前，但他整个身体都象石头一样，然后南海也因此在逐渐石化；东海之角的那位天使是半蹲着单手立掌于胸前，但他整个身体象火一样，然后东海也因此在逐渐被火化；北海之角那位神是盘腿单手立掌于胸前，但他整个身体都象在生长着一种带刺的草木，然后北海也因此在逐渐被木化；西海之角的那位天使是单掌撑地倒立着单手立掌于胸前，但他整个身体都在流出虫子，然后整个西海也因此在逐渐被虫化。

包着这四海的是一个圓形的天，整個這個圓的表面看起來象是被厚厚的白雲包住，但实际上，整个这个天圆地方又象是泡在一个紫色的火海里，这个紫色火海是由一位本事非常大的道，由它的法渗透下来形成的。这个紫色火海渗透进人体的所有细胞中，随时都在找空子给修炼人制造麻烦，随时找机会帮助大红龙势力，它们是来阻挡人得救毁灭人类的。

又有很多本事很大的、败坏了的邪恶生命从天上来到地球表面和人在一个空间，这些生命一般脚下都踩着一朵白云在人类这个空间快速穿越，速度很快，但它们体积很小，小到人的肉眼看不见，也能变大，但一般只变到象针尖那么大，这时候人的肉眼就能够看到它，它们也是随时都在找空子给修炼人制造麻烦，随时找机会帮助大红龙势力，它们是来阻挡人得救毁灭人类的。有些创世主弟子修炼出来的功能也能在人类这个空间生出一些本事很大的神，他们也很小，一般人的肉眼看不到，他们背上有神弓手中拿神剑脚下也踩着白云追杀这些来到人类空间的败坏生命。

虽然大法弟子在人中修炼被迷在其中很多事情做起来好象不如那些在暗里的邪恶那么明白，它们神通广大什么都知道，但大法弟子却有一个好处是它们没有的，就是大法弟子在人中，以人的肉身修炼出的神通法力参战的，那么大法弟子层层空间的身体和生命在这些战斗中就怎么打都是打不死的，而邪恶则都可能面临被大法弟子的功能所灭的下场。因为身在人中，如果邪恶要夺取人的生命，那必须要用人的理来实现。

这些故事，一写出来就和真正看到的那些还是不一样，真是神话故事。
