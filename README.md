# 明见地狱见闻（9）：地球海中央地狱

2020/11/24
与地球对应的冥府共有二十个，其中在中国那块地的中央垂直到地球海海底下面，就是地球海中央冥府所在。地球海中央冥府也象一个小小的世界，有城镇，森林河流，地球海中央的冥王、判官、鬼差等生活在这里。里面也有一个和中国鬼灵地狱很象的地狱，外面也是象中国古代城墙那样，方形的，但只有三层。中国冥府鬼灵市整个是建在山里面的，而地球海中央冥府是建在海底里。海底第一层象是岩石那样的结构，穿过岩石层就是土层，穿过土层突然就是一个很大的、对于里面的生命来说就象一个小世界一样的空间。在这个空间的顶部那里，有各种宝石，这些宝石有些可以制造空气，有些制造光，等等，为这个小世界提供着生存环境。

中国鬼灵地狱第六层已经是在中国那块地的最下面了，有一个出口，一条象重庆丰都鬼城无空洞那样的通道从出口出来往地球海中央地狱那个方向穿过海洋。当这条通道来到地球海一半深度的时候，海洋里有一个巨大的监狱，就象中转站一样。地球海上共有十九块地，每块地上都有一个冥府，每个冥府地狱都有一条无空洞那样的通道，都会连到这个地球海中转站。然后从这个中转站，又有一条无空洞管道一直连到地球海中央地狱那个入口。地球上所有到阴间的罪犯如果需要被押到地球海中央地狱受刑的，都会先到这个中转站等侯。

地球海中央地狱第一层里面无休止地下着一种雨，雨滴都是一块块的湿漉漉的大便，臭不可闻无比恶心，被大便雨淋着，罪犯那个难受读者应该能想象一二。大便雨下到罪犯身上后有些就粘在罪犯身上，其余大部分就会慢慢滑下来，掉落地上就立即融化。第一层地狱的行刑方法象美国地狱那样，都是鬼差动手。这些大便雨对于罪犯来说就切切实实是大便，但鬼差却觉得这些大便很香。江某某在这里受的第一个刑罚是脱水之刑。关于脱水之刑我们已经在《图谋窃取美国选举之罪》一文中说过了，就是吃一种药丸，身体的水份就会慢慢消失，最后整个身体就变成拳头那么小的一块东西。这里脱水过程的痛苦要比美国地狱里的痛苦要倍增。

地球海中央地狱第二层里面除了下大便雨之外，从这个空间的四面八方会飞出无数的回旋镖，穿透罪犯的身体，那个痛苦难以言表。回旋镖穿透的那块肉会自动愈合。这一层地狱的行刑方法也是鬼差动手，回旋镖遇到鬼差身体就会拐弯。在这一层，那个随时都被大便雨淋被回旋镖四面八方穿透的罪犯发出的各种嚎叫声都能听到，身体也能在痛苦中拼命挣扎，但怎么挣扎都无法挣脱那个锁链，最后都是被鬼差连拉带扯地弄到行刑的地方。整个场面看着又嘈又乱又恐怖残忍的样子。江某某在这里受的第一个刑罚是千刀凌迟之刑，它那个蛤蟆身仰着被固定到刑具上，那里排着密密麻麻的刀，同时往下切它的身体，也是一点点地切，切很长时间，痛不欲生。

地球海中央地狱第三层里面除了下大便雨、回旋镖之外，还会下硫酸雨，它的雨滴落到罪犯身上，就会象硫酸落到人肉体一样。被硫酸烧伤的那块肉也会愈合，就这样一边愈合一边被硫酸雨淋，这时候那个罪犯的模样就变得非常恐怖，比人想象到的那些恶鬼样子还恐怖。这一层地狱的行刑方法象中国鬼灵地狱一样都是自动的。江某某在这里受的第一个刑罚是闭气之刑，就是它全身的呼吸通道全部被堵上不能呼吸，但却又死不了。整个过程我们看到也是极其痛苦的，罪犯的身体不能呼吸后就会慢慢变老，最后变得象枯树一般非常非常的老。

江某某地球海三层地狱受刑后，就被带到南海地狱那里继续受刑。

# 明见地狱见闻（10）：网络特务与邪悟者的悲惨下场

2020/11/25
网络上有一种特务，正确来说他们应该不是真的特务，因为他们就象一个正常公司的运作，但他们做的事情却象特务一样，把自己装扮成大法弟子或者正义人士，然后在网络上骗取信任、混淆视听、带风向，更严重的是骗取大法弟子或正义人士的信任后通过邮件把木马病毒植入电脑盗取各种信息。这些网络特务以为自己躲在电脑后可以肆意妄为，其实他们的一言一行都被记录在案。

这些网络特务有专门的会议学习大法经文，研究大法弟子的用语习惯，了解当前形势变化。有时候他们直接攻击大法或者其它正义人士，有时候又装扮成大法弟子或者正义人士，到大纪元等媒体或其他媒体或自媒体的评论区写评论，如果粗略一看可能会以为他们的评论有道理，但细一琢磨，其实是能够看到他们混淆视听、带风向的险恶用心的。但现在不少人看文字时是不太爱思考的，喜欢随着自己的感觉走，所以有时候他们也能得逞，于是就会影响到一些人。

神目如电，这些网络特务的下场非常可悲，他们在拼命地把自己往地狱无生之门送。在中国东北某座城市里有一栋高楼，大楼里有专门干网络特务这种事情的、类似于公司这样的机构，里面几乎每个人都要进地狱。这栋楼二十层有一个人，他戴一副眼镜，坐在一个独立办公室里，算是这个公司的其中一个领导了。他的办公室墙上挂着一个“福”字，可是他根本就不在福中，因为他干坏事，有很多神在攻击他的背，所以他的后背总是痛。这个人乔装成大法弟子发邮件套取信任，企图用木马病毒骇入大法弟子的电脑。这个人在中国鬼灵市阎王账本上的罪很长，已经到四海中央地狱最后一层的最后了，就差一步就到无生之门。这栋楼十八层有一个大开间，很多人在里面办公，其中有一个人，他装扮成大法弟子在大纪元等评论区发表评论，粗略一看好像说的有道理，其实就是想混淆视听带风向。这个人因为干坏事，他的脚也被神攻击，所以他双脚的其中一只就总是痛。这个人的罪在中国鬼灵市阎王账本上也很长，也已经到四海中央地狱的最后一层了。写出来，希望那些网络特务能够有所醒悟。

邪悟者的下场也很可悲。有些认为自己是多高多高层次来的，是什么之主，固守他认为的那些而不能自拔；有些说某某已经修圆满或者接近圆满了，把他写的那些谈论法的内容当成经文一般；有些以为别人都糊涂了就他清醒，即使已经被明慧网点名了也还固执地干下去；有些甚至认为自己在中国反反复复转生为皇帝，历史都是他开创的，没有他中国的历史也就不存在了；有些觉得自己历尽魔难在修炼人中是万中无一的，无论他干了什么错事师父都会给他兜着；有些喜欢说自己是多高多高的认识，盗法甚至传假经文；等等。可笑又可悲！

我们看到，很多这种人一旦坠入地狱，第一个要受的刑罚就是碎尸万段之刑。我们在《图谋窃取美国选举之罪》中说过那个窃国之人要遭碎尸万段之刑。在地狱律法那里，盗窃之罪因为盗取的物品价值不同而罪业不同，其中罪业最大的可能就是盗国。一般犯了盗国之罪的，就这个盗国之罪，当他来到地狱要受的第一个刑罚就是碎尸万段之刑，身体被凌迟切成一段段。邪悟者经常犯下的是盗法之罪并非盗国，但盗法之罪之严重其实已经超出冥府审判的范围了，所以如果这个邪悟者落入地狱，他盗法之罪就只好按最重的盗国之罪来衡量。

在佛教、道教、基督教、天主教等正教之中，有些看着很有名的，或者历史上有些人们认为很有名的，其实都要在地狱里受那个碎尸万段之刑。他们喜欢把自己对法、对经文的理解安上释迦牟尼说老子说或者耶稣说耶和华说之类的名头，让人把他自己的那些理解当成释迦牟尼、老子、耶稣、耶和华说的；或者他们随意就把经书里释迦牟尼、老子、耶稣、耶和华说的一句话当成自己的话到处说，也不注明出处，使人以为这句话是他自己证悟到的，等等。如果他老是这样，盗法之罪一旦成立，这些人地狱就有名了。

当碎尸万段之刑受完后，邪悟者要受的第二个刑往往是脱水之刑。为什么第二个是脱水之刑？因为这些人犯下的第二个重罪就是传一些虚假的、不实的东西，导致一些修炼者听信了而跟着，乱了他们的修炼路，类似于美国选举中那些媒体鼓动所犯下罪行。受脱水之刑以表示要把邪悟者灌水的东西抽干之意。

佛教里有说谤佛会犯下大罪，我们看也是这样，会造下很多业。但现在是一个特殊的历史时代，大法弟子是在人的最表面修炼，要在人的最表面救度众生清理邪恶，和历史上佛教基督教道教等正教修炼形式都不同，有些大法弟子就办了各种媒体。就我们极其有限的层次看到的情况来说，人类社会里任何一个组织、或者媒体、或者公司、企业等等在另外空间不同层次有具体的生命体现，有些体现出来的可能是魔，有些可能是神，而共产党共产主义体现的就是约翰《启示录》里说的那条大红龙。就我们层次看到的，《大纪元》在另外空间体现出来的是一位光耀无际的佛，毁谤大纪元和谤佛所犯之罪差不多，就这么严重。

邪悟之人喜欢毁谤明慧网，或者到处对人抱怨明慧网有什么什么问题，他们的借口之一是明慧网也有错的时候。其实在我们看来，如果你觉得明慧网有错那可以向明慧网发邮件阐明观点，就只能到此为止。如果为此愤愤不平，或者因为明慧没有回信或者没有同意或者被明慧批评点名等等，而到处发泄私愤，甚至攻击明慧，则毁谤明慧之罪就会坐实，在另外空间看，这个后果极其严重，比谤佛还严重。就我们极其有限的层次看到的情况，我们看到人类的这些媒体、组织、公司、或者什么主义等等，或者人类空间这里的一块砖头、木头等等，就我们层次了解到的范围之内，在另外空间体现出来的都是一个具体的生命，但明慧网除外。我们看到明慧网在另外空间体现出来的就是五个字，在我们有限层次范围内看到的层层也都是这五个字。具体这五个字是什么我们也看不清楚。就我们个人层次的有限理解，明慧网的威德应该是不能用“生命”来衡量了。

写出这些，是希望邪悟之人能幡然醒悟，别因为执著自己的那点东西而失去了万古机缘。

我们明见神话写出的所有内容都是神话，不一定都是真的，也更不一定对。信者自明，不信者且当神话陶冶性情！
