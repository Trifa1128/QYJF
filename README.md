# 七印解封（26）：诺亚方舟是真的

2020/09/25
因为诺亚方舟在西方文化中占有比较重的分量，所以我们在讲中国人神共通这件事情之前还是先把诺亚方舟这个事情说了。

诺亚方舟的故事来自于圣经《创世纪》，大意是说耶和华见地上人类道德败坏，于是计划用洪水消灭恶人，然后他发现人类之中有一位叫做诺亚的好人，于是他就指示诺亚建造一艘方舟，并带着他的妻子、儿子与媳妇以及牲畜与鸟类等动物雌雄一对对的带上方舟。我们从七印书卷看到，这个诺亚方舟是真的，诺亚这一家人也是真的，很多牲畜和动物被带到方舟上也是真的，但是用洪水毁灭人类不是耶和华的主意。

地上人类涉及到东西方佛家道家两大体系，所以涉及整个地球的变化不是哪一家单独就能决定，以洪水毁灭上一次人类文明是当初制定七印书卷的那些神定下的。耶和华知道人类社会要发生的事情，而对于下一次人类文明要做的事情耶和华也早有计划。我们在前面关于本次人类文明时期犹太人的历史已经提到，本次人类文明犹太人是被耶和华拣选来配合耶稣留下一段关于救赎的历史的，是要为创世主预备道路的。所以在看到人类要被大洪水灭世，耶和华就需要留下一些犹太人为即将开始的新的文明历史准备，于是他就拣选了诺亚作为留下犹太人的关键。当然诺亚能被耶和华拣选也是因为诺亚本身对神的信达到了那个程度。耶和华是以梦境的方式启示诺亚的，诺亚在梦中看到有神告诉他不久人类就要被洪水毁灭，要他造一艘巨型的船，其中要有一层是给动物准备的，诺亚醒来后相信这个梦是真的，于是他就按梦中所见到的那艘船开始了庞大的造船计划。

诺亚一家那个时候是住城外，在当时属于比较有钱的人家。这个城按今天地理位置来看应该在土耳其迪亚巴克尔和尚勒乌尔法之间。当地有一些人也相信了诺亚的话，于是加入到方舟建造中来，但人手还是远远不够，于是诺亚就出钱请了很多人来造方舟。这个方舟不是在水里造的，它是在陆地上一块开阔地方造的。方舟外型上就是长方形的，船底也是平的，大概有现在一个半的标准足球场那么大，共五层，每上一层就稍微比下一层小一些，每一层也是方的。其中第一层是储存间，第二层是给动物准备的，第三层是用来做饭吃饭活动等等之类的，第四层和第五层是用来住人的。方舟大约四个月就造好后，神通过梦境指示诺亚马上让人搬到方舟上，因为洪水马上要来了，于是那些不相信的工人拿了工钱就回家了，而那些相信的就立即把自己的家人、家里的家禽家畜之类的，等等都带到船上，远远近近也有些人也都赶过来，然后第二天很多动物就从树林里走了出来，就象人很懂事一样上到第二层神在梦境中吩咐诺亚为它们准备的地方，其实这些动物在方舟造好前的几天就已经陆陆续续在神的指引下来到外面的树林里等着，只是那时候人们没发现。方舟造好两天后，灭世的大洪水就来了，当时上到诺亚方舟上的人大概有三千多。

人总以为再大的天灾也会有预警，人也能提前做好准备逃生，其实人类毁灭的方式很多时候都不是当时人所能够提前预警的，也不是慢慢来的。故事说这场灭世洪水是因为全球连下了四十天的暴雨导致的，我们没看到这个情况，这个灭世洪水是在地球上突然出现的，人根本就来不及逃生。我们从七印书卷看到，当灭世的时辰一到，全球各地就来了约176位神，其中约100位是属于佛家体系的，76位是属于道家体系的，他们按不同方位分布在全球各大海洋湖泊的上空。只见这些神打出手印，然后在他们手印的下方就出现了一个金色的圆形的字，他们转动这个字，这个圆形的字就象一把钥匙一样，下面的海水或者湖水就会分开一个圆洞直达海底或湖底，然后地下就裂开一个口。当这个口一裂开，那些神就立即飞走了，然后就看到全世界各地从那个口里出来无数的水，就象是突然从土里冒出来一样一下子就把地球淹了，这个速度之快，那些地面上的人根本来不及自救。

这些水的来源我们也看了一下，是众神从南瞻部洲所在的那个南海里把差不多一滴的水珠放进了地球。

诺亚方舟被那些突然出现的大洪水托到水面上，耶和华用他的神力给这个方舟下了一个罩把方舟保护起来，以免它因为颠簸或者漏水而翻了，这个罩诺亚他们看不到。如果不是耶和华的保护，这艘方舟早就七零八碎了。圣经故事说洪水淹没了世界最高的山，我们也没看到这个情况，世界上不少高山并没有被淹没，但因为洪水来的太快，世人根本就来不及往山上跑，只有那些原本住在山上的，或者听信了神的启示而提前跑到山上的才活下来。诺亚方舟后来飘到了亚拉腊山，是停靠在这座山半山腰往上一些的地方，因为半山腰往上就没被水淹了，并非圣经故事描述的山顶上。那里当时山上也是森林，于是船上的人和动物全部都离开船去到山上等待洪水退去，他们也在山上盖房子住下，这一等也象中国人一样差不多一年。而当洪水慢慢退去时，那些退去的洪水也把诺亚他们造的那艘方舟慢慢拖到海里沉下去了，所以其实这个方舟作为一个整体目前在地球已经不存在了。于是，本次人类文明开始后，犹太人就首先出现在土耳其。圣经关于洪水、方舟的故事是真的，但里面很多情节就有问题，甚至说诺亚活了多少百岁，我们看到他是一个普通人，没超过普通人活的岁数。

当时不仅诺亚他们，整个地球上不同民族国家的人们都得到了他们的神用各种方式启示他们的人，告诉他们即将到来的灭世洪水，有人相信，但绝大部分都是不相信的，那些相信的就往高山上躲避，那些高山上幸存下来的人就成为了本次人类文明各民族的祖先。而在中国也有很多神带来的启示，也有很多地上的修道人出来告诉当时人们这个真相，那些相信的就往青藏高原那里的高山跑，于是那些高山上幸存下来的人就成为了本次人类文明中国人的祖先。而且在大洪水来之前，地球上那些没有被洪水淹到的高山上本来就有很多动物，当大洪水出现后，那些本来在山脚的一些动物也都跑到山上来了，所以，本次人类文明时期地球上的动物大部分都不是从诺亚方舟那里来的。

# 七印解封（27）：打开人神共通之门

2020/09/28
人类历史关于大洪水灭世的传说只说了地球上的人类文明被毁灭，其实每一次涉及整个地球的灭世事件，不仅仅是毁灭地球上的人类文明，在南瞻部洲上对应着地球的那块地上的那些生命也同样在劫难逃。我们在《南瞻部洲（5）：宝塔神兵》中提到过，南瞻部洲那个地上面有一个很大的海，它就是地球在那个空间的体现，而那个海里有十九块地，地球上所有国家都在这十九块地上。我们从七印书卷看到，当地球被众神从南海那里调了差不多一滴水珠把地球淹没的时候，在南瞻部洲这十九块地的上空也下起灭世的大雨从天而降，那个雨是那里的生命从未见过的，象子弹一样直击而下，迅速就把这十九块地淹了，而那些能飞的生命如果想飞出去，如果是注定要死的，就会被空中的雷电击中。人类空间留下了一些生命，那个地方也同样留下了一些生命。

中国这个地方在上一次人类文明时期是道家文化盛行的时代，文字也是象形文字，但在上一次人类文明时期是没有纸的，是用专门的刻刀刻在专门制作的片形物品上，或者用笔写在上面。如果拿今天的人类生活方式和上一次人类文明被洪水灭世时候的生活方式相比，那时候人类生活还是很单纯的，没有今天互联网给人带来的那样混乱的思想，也没有现代这些令人眼花缭乱的智能设备和各种千奇百怪的夜生活，感觉现在的人类社会如果按那时的灭世标准来衡量早就应该被灭世好多次了。

当初在昆仑山等高山上存活下来的本次中国人祖先，他们虽然思想中有上次文明时期各种知识，但因为物品的制作是一个需要社会整体技术相互配合的产物，就象今天的芯片制造一样，很多国家、那些学这一门的老师学生都知道怎么造，但如果没有那些相互配合的工艺和产品，这个芯片他就造不出来。大洪水把上一次文明时期所有用来制造物品的机器之类的全部毁灭了，所以那些存活下来的他们也是经历了一段比较艰苦的生活，是在什么都没有的情况下从新开始的，所以虽然也能盖房子，但已经和上一次文明末期盖的房子相比不一样了，造一个火折子也不是那么容易了。然后随着一代代繁衍，因为很多关于上一次文明的那些知识也都用不上了，于是很快被后人遗忘。

但这也是必然的，因为每当新一期文明开始后，上一次文明的知识很多就被神有意的不让它们出现在新一期文明历史中，以免对新一期文明时期所要塑造的文明产生干扰。而且，无论东方还是西方，上一次人类文明幸存下来的人，他们被安排的使命其实不是来开创新一期文化历史的，他们的使命是被安排来繁衍出新文明所需要的那些人以及准备一个社会环境。这段繁衍人准备社会环境的历史大概是五百年，东西方都差不多是这个时间。这些幸存下来的人们在这五百年的历史中也有非常多的精彩故事，而且很多事情看起来也是非常了不起的，但因为他们的这些事迹不属于新文明时期需要的，于是也都不被记录下来。本次人类文明在大洪水灭世后五百年正式开始，也就是距今约四千五百年前，所以司马迁写《史记》的第一章就叫作《五帝本纪》是很对的，而后人在前面加一个《三皇本纪》就显得有些画蛇添足了。我们这里特别强调一下，中国历史所说的三皇那些人都是有的，但他们是史前某一个历史时期的历史人物。

因为青藏高原的气候变得越来越不好，这是其中原因之一，当然也还有其它原因，反正就是很多中国人就从那个地方迁移到各地。那时候还没有人称王，地大物博，你想去那里就去那里，也没人管。一起离开的一般都是有血缘关系的，或者关系好的，大家一起搬到一个新地方开始组建新的生活，所以中国人历史说氏族那也是很对的，就是很多个氏族散布到中国各地。另外，当时的社会无论哪个氏族，都很相信修炼的事情，当时大概一半的人是普通人，然后另一半的人都和修炼有关，他们或者入世学道练武学文，或者出世修道。出世修道的人到处都有，也有在上一次文明时期就修炼的高人出来世间行走。我们从七印书卷看到，那个时候的社会出来很多有功能的、开天目的人，可以说整个社会都处在这样的道家文化之下。在中国历史上有个人神共通时代的说法，于是不少人就认为这就是人神共通时代了，其实还不是，这只属于整个社会道德较高很多人都相信修炼也有很多人修炼而带来的整个社会表现出来的状态。

中国所出现的那个人神共通时代，是因为中国那个地方被打开了一条通往四大部洲的通道，然后中国那里的普通人用脚就可以走到那边，那边的生命也是用脚就可以走到人间来，是这么一个特殊的历史时期。在本次人类文明时期古希腊也出来了很多神话故事，似乎也有一个人神共通的时代，但那是无法和中国所处的这个相比的。当初在古希腊为了留下神话，只是让一些特别的人进入了另外空间一个非常有限的范围。但在中国那里却是普通人就可以去，甚至有不少的人是游遍四大部洲的，而且很多人都能够到《山海经》所说的海内经的范围之内，那边也有很多生命来到中国。如果《山海经》上面写满了那些曾经从南瞻部洲进入中国的生命物种的名字的话，那些名字装满一屋子都不止，绝非《山海经》所写的那一点。

中国本次人类文明出现的这个人神共通时代，是本次地球亿年历史从未发生过的事情，这是头一回，是为了创世主带着他的众多授记弟子从昆仑山之顶转生中国那个地方留下神传文化而特别打开的。

我们从七印书卷看到，当历史来到约四千七百年前，也就是创世主从昆仑山顶临凡之前的两百年，三界之外很高层次上有位道，他是穿白色道服拿白色拂尘的，创世主给了这位道两个指环一样的法器，每个指环有两个孔戴在两个手指上，然后这位道就来到三界最低层四大部洲的上空。只见他双手这边划那边划，于是就看到四大部洲从原来那个位置移到中间连接在一起了。我们在讲《南瞻部洲（1）：天圆地方》说过，在四海中间用来划分四海的那个纵横分界线的交汇处是魔王所在，它象巨大的火山口那样的存在，海水进不来，里面深不见底，当这四大部洲被这位道移到这里后，四大部洲的交汇处就覆盖在这个火山口上。我们知道，原来四大部洲都是被四个大海包围着的，四个大海之大，不是四大部洲里的生命随便就可以飘洋过海的，所以四大部洲的生命几乎是没有往来，除非有特殊能力，但现在四大部洲连接起来了，这对于当时生活在四大部洲的生命来说也都是一件天大的事。虽说四大部洲连一起了，但实际上真正能够用脚从这一部洲走到另一部洲的就是南瞻部洲、西牛贺州和东胜神州，北俱芦洲虽然也连在一起了，但一般生命也还是去不了。

当四大部洲连在一起后，这位道就来到中国吕梁山脉所对应的地方，然后双手一划，就打开了一条中国与四大部洲相通的通道。这条通道不是一个洞，而是整个吕梁山脉都打开了。

# 七印解封（28）：礼仪之邦的来源

2020/10/02
吕梁山脉一带在今天中国历史可能不属于什么名山大川，但在那个时候，因为人神共通从这里打开，那是远近闻名了。这个地方打开后，中国这边的人如果走到这个地方，他其实已经踏在了南瞻部洲的地上面，但他可能还是以为自己走在之前的地上，走了一会，他就会看到很多奇怪的动物，是之前从未见过的。

《山海经》里面记录的就是南瞻部洲地上面的山川河流的事情，它不是大禹写的，是人神共通之后那时候的人记录下来的，当时记录下来的很多，然后流传下来的这些东西在大禹治水的时候又从新整理了一遍。但后来很多都失传了，今天人们看到的《山海经》只是当时记录下来的一小部分，而且后人打开《山海经》方式也不对。我们知道现在《山海经》第一篇叫南山经，这就已经错了，其实正确打开的方式是“中山经”。中山经说的“薄山”是南瞻部洲上对应到“吕梁山”的那座山。因为南瞻部洲很多事情是和人间对应的，那些山也差不多，人这里有条河，它那里也有条河，比如中山经说“共水出焉，而西流注于河”，那条河就是对应到中国的那条黄河，那条河的河神就是我们在“长江黄河另外空间所见”说的那位。中山经说薄山之首曰甘枣之山，它那里有一种兽叫nuo，我们看到那个地方确实有这么一种动物，它的耳朵嘴巴和身体大小象松鼠但身体和头上、脸上的纹路都象老虎，如果把这种动物的心入药是能够治好瘿病。

因为本次文明开始的前五百年是为本次文明历史的开创准备人和社会环境的，所以到人神共通开始的时候，中国人的繁殖能力是很强的，出生的小孩一般都能活下来，于是也就是到约四千七百年前，中国已经到处都有人，组建出很多部落，北边到蒙古，东北到黑龙江、朝鲜，南边到广东福建，西边到广西云南，到处都有从新疆西藏高原那一带迁移下来的人。而这些人无论在那个地方，东北也好，还是中原地带河北河南，还是被称为南蛮之地的两广云南，人们穿的衣服和住的房子在那个时候都差不多，所用的技术也都差不多，都是从上一次人类文明留下来的文化体现。

那时候人们穿的衣服并非有些书上写着说披着树叶裹着兽皮，其实也都是很好看的衣服，但不是用今天这些化学工艺造的，他们是从一种树叶那里抽丝出来的，也有从树皮那里抽丝出来，而今天人们知道的麻布衣就是其中一种，但从树叶抽丝出来做成的衣服现在似乎看不到，估计这种技术已经失传了。另外，武士都有佩剑，也有盔甲，但盔甲不是用铁造的，而是用动物的骨头把它融化了添加一些矿物质然后按照身体需要保护部位的形状凝固成形，那个盔甲很轻也很坚固，也有象诸葛亮七擒孟获里说的藤甲。

其实今天中国广西云南那里的少数民族保留了一些上一次人类文明的文化，例如他们的服饰和房子。上一次人类文明末期中国这个地方的房子也象广西云南那些少数民族的房子，也是一盖好几层，没有中国四合院那种讲究，衣服上饰物的味道也似乎是那个样子。当然，现在广西云南看到的少数民族的房子比人神共通开始前中国人盖的房子要有规则的多，那时候可能是因为受道家修炼文化影响，我们看到他们盖的房子是没有太多讲究的，盖成圆的也有，盖成方的也有，不方不圆的也有，围墙弯弯曲曲的也有，反正就是个人喜欢，而穿衣服上也不是太讲究对称匹配，武士的盔甲有时候经常左右不对称，就是都好像比较随意，人与人之间也没有太多礼节上的讲究，例如见面打招呼，有很多人就是闭着嘴用力吹气那样。

但所有这些都在人神共通之后彻底改变了。人神共通对中国的最大影响是给中国人带来了一种从未有过的礼法。我们知道中国历史上被称为“礼仪之邦”，人们的穿衣打扮、言行举止、吃喝拉撒、房屋建造的考究，以及王侯将相士大夫的礼制、将军士兵的盔甲旗帜、行军布阵，等等，都有完整的、非常讲究的“礼法”，它们从哪里来？其实它们不是谁发明出来的，或者是经过了一个漫长的历史人们自己形成的，或者哪位神仙下凡传给人的，都不是，这个“礼法”它来自于南瞻部洲上面那个中国城。

南瞻部洲上面的生命不象人间那么迷，那里的生命知道玉皇大帝，知道下雨是龙王在上面施雨，也经常可以看到天上的神，不仅如此，本身那个地上就有麒麟、凤凰这样的神兽神鸟，也有很多天生就有特异功能的人，不少生命可以来往于天地之间。反正是和人间完全不同的一个世界了。那个“礼法”在南瞻部洲上面是必然的，就是那样的，就象天宫在地上投影一样。所以当土里的中国人到了这个地方，那是被惊呆了，美不胜收，天上宫阙，反正就是全部把南瞻部洲上有的，只要能学到的都搬到人间来，于是土里的中国人在历史上就象突然就有了一整套体系严密象天上一样的礼法，有了皇帝要戴什么帽，皇宫应该怎么盖，文臣武将应该怎么样，人与人之间应该怎么个来往，男人衣服怎么个穿法女人衣服怎么个穿法，怎么行住坐卧，等等。于是，在土里埋着的中国就出现了本次地球亿万年历史以来从未有过的“礼法”，把中国这个地方搞的象天上神仙一般。



当然，人神共通的通道能打开，也是因为土里中国人整体上处于一个道家修炼这么一种状态，有很多修炼人，也有很多出了功能神通的，而且当时中国人学的武术也是非常厉害的，就象历史书说的，有些人能擒龙灭怪，李元霸力大无穷，那时候也是有的。而且人类是一个特殊的空间，虽然土里的人出生时无法和南瞻部洲地上面的人相比，那上面的生命出生用人的语言来说几乎都是“生而神灵”，所以对于普通人来说就本事比较大，骑个麒麟也不是什么很难的事。但被埋在土里的中国人通过修炼就很快超过他们，所以土中的中国对于南瞻部洲上面的生命来说也很神奇，也有很多那边的生命来到人间，于是大家就这么相互来往。人神共通时代刚打开，无论对于南瞻部洲还是对于土里的中国人来说都无比的惊奇，但久了也就习惯了，慢慢也就觉得本来就是这样的。

另外打开通道之后那位道他就要一直在那里不能离开，他要一直负责着，做很多事情，这样两个不同空间的生命才能够相互来往。

不久的将来人间是否也会出现一个这样的人神共通时代？我们很期待！

当初吕梁山下有两个氏族，一个在吕梁山西边靠南，一个吕梁山东边靠北，西边靠南的氏族称为神龙氏，东边靠北的称为烈山氏。中国人说“近水楼台先得月”，这两个氏族最靠近人神共通的通道，而那些远近氏族要想去南瞻部洲地上面都要经过他们的地方，于是，神龙氏和烈山氏很快就成为当时整个中国最有影响力的两个氏族，其中神龙氏的统治范围一直到新疆那边整个中国的西部，包括西藏、青海，也包括蒙古，烈山氏则包括河北、河南、湖南、湖北、山东、安徽等东南部。那时候的西北不像现在那样，也不是象春秋战国秦汉所描写的那样是少数民族居住之地，那时候是当时中国最发达的地方。而后来被称为南蛮之地的两广、云南等地，可能是因为离吕梁山较远，或者是神有意而为，反正他们和人神共通的事情似乎关系不大，不在神龙氏和烈山氏范围之内。

当历史来到距今约四千五百年前，象神话一样的社会环境都已经备齐了，创世主就带着他众多的授记弟子从昆仑山之顶下世，开始了为人类的得救开创辉煌的中国神传文化的征程。

# 七印解封（29）：黄帝学艺

2020/10/04
创世主从昆仑山之顶带着众授记弟子转生中国后，创世主当时转生为黄帝。史记《五帝本纪》说“轩辕之时，神农氏世衰。诸侯相侵伐，暴虐百姓，而神农氏弗能征。”确实是这样。从人神共通到黄帝出生，历史过去了约两百年，在中国这片土地上大概形成了约136个大大小小的诸侯国，而当初最强大的神龙氏已经无法臣服它原来所瞎管的各大诸侯，而且很多诸侯本身的国土面积也都已经和神龙氏差不多，有些甚至超过了神龙氏，这些诸侯互相打，就象后来的春秋战国那样。但当初和神龙氏一样强大的烈山氏经过这两百年的发展不仅没有衰败，反而更强大了，于是烈山氏就想着征服原来属于神龙氏的那些诸侯国。黄帝就是在这么一个历史背景下出生的。

历史上关于黄帝的记录不多，也比较乱。《国语·晋语》说：“昔少典娶于有蟜氏，生黄帝、炎帝。黄帝以姬水成，炎帝以姜水成“。我们没看到黄帝和炎帝是两兄弟，我们只看到炎帝属于吕梁山以东的烈山氏，黄帝属于吕梁山以西的神龙氏。史记《五帝本纪》说：“黄帝者，少典之子，姓公孙，名曰轩辕。生而神灵，弱而能言，幼而徇齐，长而敦敏，成而聪明。“我们看到，黄帝出生在陈峰氏这么一个部族中，当初这个部族属于神龙氏的一个分支，有国号，应该相当于诸侯，面积不大，后人称之为少典，但我们从七印书卷看到这个字很象沙字，也就是“少”字前面有一个三点水，而典字前面也有个三点水，但为什么后人把三点水去掉我们不是很清楚，我们就按照现在历史的说法叫少典。《五帝本纪》说“少典之子”也没错，因为黄帝的父亲是这个少典的国君。少典国的位置大概在今天陕西省延安市和榆林市之间的某个地方。

当黄帝长到五岁的时候，他的父亲少典国君就张榜设下三大关为黄帝招聘启蒙老师。第一关要打败少典国最强大的武士，第二关要破解少典国最厉害的阵法，第三关要博闻强记。来应聘的人很多，但没有一个能够通过这三关的。这一天，黄帝家来了一位应聘的，他长得有些象三国演义图书画的那个老了的邓芝，是在横断山脉里修炼的一位道人，他是专程为黄帝而下山的。第一关，那位道人来到比武场上对着少典国最强大的武士拱手说了一声“请”，然后那个武士就使出浑身武艺攻向道人，但大家都没看到道人怎么出手，反正就是武士刚要使出招数就被击倒了。然后第二关，少典国将军拿出一个象围棋盘那样的器具，里面摆了一个他们认为最厉害的阵法，这位道士看着阵法就大笑说这也太简单了，然后举手间就把这个阵破了，跟着这位道士在上面也摆了一个阵法让少典国将军看，少典国将军只看得奥妙无穷无法破解。第三关，少典国见识最广的一位智者拿出一本古老的书要这位道人先读一读，然后他会根据这部书的内容提问题，但道人却说这部书不用读了，你就直接问吧。结果无论那位少典国智者问什么问题，这位道士都能回答。于是，黄帝父亲设下隆重的拜师礼，让黄帝拜这位道为师。

这一学就学了七年，黄帝十二岁的时候，有一天这位道来见黄帝父亲辞行，说自己这七年只是教了黄帝一些基本功，黄帝还需继续拜师学艺，然后交给少典国君一封信，让黄帝带着这封信去某个地方找一位高人学艺。这个地方在今天看大概在山西运城一带。少典国君就按那位道人的吩咐，给黄帝准备了一匹好马，带齐各项物品，择了一个吉日就让黄帝只身一人远赴他乡学艺去了。

黄帝马不停蹄地按照道人所说来到了一座城，城里有很多人，很繁华，他也不知道自己要找的那位高人在那里，城里的人都不知道有这么一个高人。黄帝就一路问来到一个铸剑的地方，那里挂着很多宝剑售卖，里面有一个白胡子老人，这些剑都是他打造的。黄帝就问这位白胡子老人，问他是否知道某某高人，那位老人就说“知道”，黄帝大喜过望，就问是否可以告诉他，老人说“可以，他在”，然后老人把脸朝向左边，黄帝立即着急地用手指向左边说"哦，是在那边"，老人说“不是”，黄帝又立即着急地用手指着右边说“在那边”，老人笑着对黄帝说“你不要着急，你要静下心来听我说”，于是黄帝只好按捺住性子，然后老人说，“你要找的人就站在你面前”。黄帝大喜过望，连忙把第一位师父的信交给这位老人，老人看完后就收了黄帝做徒弟，教授黄帝驾驭水火的道术。于是黄帝就跟着白胡子老人一边铸剑卖剑一边学习道术。

这一学就学了四年，黄帝十六岁的时候，已经学会了驾驭水火的道术，水不能溺火不能焚，也能踏水而行，也能口中吐火口中吐水。这一天白胡子老人对黄帝说你已经可以出师了，但你还需要到某个地方找一位高人跟他继续学艺，然后就给黄帝一封信和一把神剑。这把神剑可以变化，不是用人间的铁打就的。《中山经》里描述了南瞻部洲上面有一座山叫鲜山，山中有一种动物叫鸣蛇，状如蛇而四翼；这把神剑的剑身就是用这种鸣蛇的皮做的，而剑柄则是南瞻部洲上另一种动物身体里的一种物质提炼出来的。于是黄帝就带着信和神剑去寻找他的第三位师父了。

他的第三位师父大概是住在今天河北石家庄那一带，那里当时也是一座城市，人口很多。黄帝来到这里后找了很多天都没找到这位师父，那里的人都不知道有这么一位高人。这一天黄帝看到前面有个肉铺，于是他过去问卖肉的是否认识这么一位高人，那卖肉的说知道，黄帝很高兴，就打躬作揖请问高人住在何处，卖肉的就详细地对他说你往左走，然后看到什么就右转，然后看到什么就左转，反正告诉了一大段话。黄帝听完拔腿就跑，按照卖肉的所指的路线一路走来，结果发现自己又走回到原地看到肉铺，很奇怪，就问难道走错了我怎么又回来这里了？卖肉的就笑着说，你面前的就是你要找的那个人啊。于是黄帝就把第二位师父的信交给这位卖肉的高人，然后这位高人就收了黄帝做徒弟，于是黄帝就一边卖肉一边学习各种拳脚刀枪剑戟等真功夫。这些真功夫可不是当时社会上将士所用的，看起来象是天兵天将所使用的，例如其中一种剑术，黄帝双手各执一把剑，人就飞起来了。

这一学就学了五年，黄帝二十一岁的时候，这一天第三位师父告诉黄帝说你可以出师回家了，去做你该做的事情。于是黄帝就带着满身本事辞别师父回到少典家中，然后不久少典国君就对他说我们现在神龙氏面临很多问题，内部诸侯互相攻击，外面烈山氏虎视眈眈，你既然学了满身本事，那就去求见神龙国君，出一份力。于是黄帝就离开了少典国，来到了神龙氏都城所在。

# 七印解封（30）：黄帝获麟

2020/10/07
黄帝为了历炼自己，他离开少典国去神龙氏都城是走路的，没有骑马。神龙氏都城一开始在吕梁山之西靠南，后来在黄帝出生前搬了一次都，搬到了今天内蒙古鄂尔多斯一带，那时候那一带地貌和现在不同，那些地方都是肥沃之地，到处都是森林、河流湖泊，人口众多。我们估计神龙氏搬迁也可能和烈山氏势力越来越强盛有关，因为这时候烈山氏已经占据了河北、河南，山东等地，今天陕西运城那一带也属于烈山氏，离神龙氏原来都城很近。

这一天黄帝来到一个客栈住下，他晚上打坐时体察出附近有异常，于是他右手握神剑，凝神定气。屋里突然出现一个黑影在他周围快速跳动，速度很快，然后只见黄帝神剑出手，跳动那个身影的脖子就被神剑象一把绳索一样套住，黄帝一看原来是一只猴子，他立即左手一掌把猴子击飞，然后右手一剑刺向身后，只听啊一声惨叫，从黑暗中一个人中剑倒地，跟着出来一队武士攻击黄帝，但敌不过黄帝武艺高强，这些武士就败退逃走了。被黄帝神剑击中的倒地之人并没有死，于是黄帝就为他疗伤，治好了他。这人很感激，黄帝问他叫什么名字，他说自己没有名字，于是黄帝就赐了一个名给他，为了行文方便我们这里把他叫“黑者”。

那时候中国和现在很不一样，有很多各门各派的练武之人，都是真功夫，不是现在那些花拳绣腿。在一个原始森林里有一个练武之人，他所练的很象今天知道的日本忍术，我们估计日本忍术可能就来源于他那里，具体怎么个来源我们就不细查了，因为查一个事情的来龙去脉真是很麻烦的一件事情，我们就把他这一门称为忍术算了。他带了一帮徒弟，都是无名无姓，象忍者一样生活，他们所学的很多都象日本文化里说到的各种忍术。黄帝身上他第二位师父送给他的那把神剑可以说是世间至宝，这个忍者他有一天看到了黄帝身上的那把神剑，他就动了邪念想得到它，这些偷袭黄帝的就是他派来的。我们也看了一下，黑者右肩膀上有创世主的授记，他其实是追随创世主从昆仑之顶下来的。这个黑者在历史书上没有提及，因为他跟随了黄帝后就作为黄帝的贴身护卫了。

黄帝带着黑者离开客栈就一直往北走，这一天来到一个镇上，有很多人，当他们走过一个肉铺的时候，听到卖肉的喊道，“卖肉啊，有猪肉、羊肉、牛肉，还有麒麟肉、寓鸟肉”。黄帝听到了“麒麟”，黑者听到了“寓鸟”，两人都大吃一惊。麒麟和寓鸟都是世间罕见之物，怎么在这里被人做成肉叫卖？因为黄帝第三位师父就是个卖肉的，所以黄帝心想或许这位卖肉的也是高人，于是就请教卖肉的，那位卖肉的笑着说我这里没有麒麟肉，也没有寓鸟肉，但如果你们想要，我知道那里有。黄帝听师父说过将来如果有机缘一定要以麒麟为骥，而黑者则听他的师父说过如果忍者能得到一只寓鸟那是忍者最好的护甲了，于是黄帝和黑者几乎异口同声请求这位卖肉的带他们去。卖肉的也很爽快，马上就把肉铺关了，带着黄帝他们七弯八拐的来到一山中。其实这个卖肉的就是一位高人，他是专门在这里等黄帝的，喊卖麒麟肉、寓鸟肉也是因为看到了黄帝就故意引他们。这位高人对黄帝说这山里就有麒麟，问黄帝知道怎么驯服麒麟之法吗？黄帝说不知道，然后这位高人就教黄帝怎么驯服麒麟之法。于是黄帝和黑者就走进了山中。具体山中发生了什么事情我们就不细说了，因为这样细说下去我们就不知道什么时候才能写到那个终局了。反正过了一段时间，黄帝就骑着一只紫色的麒麟从山里走了出来，而黑者肩膀上也站着一只寓鸟。

麒麟按能力来分大概可以分为三类，一类叫地麒麟只能在地上跑不能飞，第二类可以称为飞麒麟因为它可以飞，第三类可以称为天麒麟因为它可以变。黄帝得到的这只麒麟就属于天麒麟，南瞻部洲地上面都是罕有的，更不用说埋在土里的中国了。每种麒麟怎么收服它也是很有讲究的，都有一套收服的方法，黄帝从那位高人那里知道了怎么收服麒麟之法，而怎么收服寓鸟，黑者的师父也教过他，因为这是忍者梦寐以求的。《山海经》北山经说有座虢山，山上很多寓鸟，状如鼠而鸟翼，其音如羊，可以御兵，说的就是它。《山海经》书上说能御兵但怎么御兵没有说，其实这种鸟身上能够发出一个能量罩，把那些攻击的兵器挡在外面。

那位高人很高兴， 就问黄帝你要去哪里？黄帝说要到神龙氏都城，高人就说从这里到都城有五条路，每一条路上都有拦路的，不好过，你们想走哪条？黄帝问怎么个拦路法？高人就说第一条路上有一个火人，第二条路有一个怪兽，第三条路有一个功夫很高的人，第四条那里有一个很大的悬崖要飞过去，第五条路有一位道人他会出五道难题。黑者马上说走火人那条路，因为黑者懂忍术之中的火术，他的嘴能吐出火，而黄帝也同意，因为黄帝第二位师父教了他御水火之术。于是那个高人用手一指，那你们就往这个方向走。

可是，当黄帝他们按照高人所指的走着走着，他们发现前面路中间坐着的是位出五道难题的道而不是火人。黄帝知道这一关自己是必须过了，骑麒麟飞也肯定是飞不出这个道的手掌心，只好前去向这位道打躬作揖，问这位道自己要从这里过。那位道说你要想从这里过就把你身上那把宝剑给我，或者你们能回答我五个问题，如果第一个问题回答不出来你们就要被鞭打一百鞭然后回答第二个问题，如果第二个问题回答不出我就要从你们身上剐去一块肉然后回答第三个问题，如果第三个问题回答不出来就要把你们放进油锅炸半个时辰，如果你们能活下来就回答第四个问题，如果第四个问题回答不出来你们就要被火烧半个时辰，如果你们能活过来就回答第五个问题，如果第五个问题回答不出来，我就要取走你的宝剑。黄帝问怎么个五道难题，那位道就用手一指远方说你看那边那棵树叫什么名字？黄帝一看那颗树不是人间有，但好在他第一位师父博闻强记，关于这种树的知识也都教过，于是就笑着说这棵树叫什么什么。于是那位道接着出第二道问题，指着远处的森林说你们要在一招之内把那边五十棵树都砍下来，黑者说这个我来，然后从身上取出一把象飞镖那样的武器一下就把五十棵树砍倒了。于是那位道说你们要给我表演一套人间没有的武艺，黑者不知道这是什么武艺，但黄帝知道，因为他第三位师父教的都不是人间武将所用的武艺。他叫黑者把他背上的两把剑给他，于是黑者从背后取出两把剑交给黄帝，黄帝手拿双剑武了一套天兵剑法。于是这位道就把拂尘往空中一抛说，你们在一炷香的时间内要抓取这个拂尘，黑者一见就立即说抓东西是我的看家本领，然后上前一抖身很多暗器之类的就从身上飞出来击打拂尘，但拂尘好像长了眼居然避开了所有这些暗器，于是黑者施展他的追踪术追击拂尘，但无论怎样也追不上，眼看时间要到了，黄帝叫黑者退下，然后从腰上取出神剑抛向拂尘，神剑变成爪一样飞出去就把拂尘抓住了。于是那位道就说最后一个问题是你们要打败我。黄帝只好和黑者一起围攻这位道，但怎么也打不败他，黄帝只能认输。那位道就说那你身上的神剑就只能归我了，黄帝虽然百般不愿意，但也没办法了，只好解下宝剑递给这位拦路的道，然后这位道就放他们离开了。

我们把这个事情的因缘看了一下，原来这位道就是那位卖肉的高人，他是一位修道之人，当他修到一定层次将要往更高层次突破的时候，他看到一位神，这位修道人在某个历史时期曾经欠过属于这位神的债，这位神要这位道还了这些债才会放他提高到上一个层次，还债的方式就是这位道要生活在这个镇里收集一定数量的宝物给他。当黄帝路过这个地方时，他那个债主就说黄帝身上那把宝剑抵消后面所有的债，于是这位修道人就先让黄帝得到麒麟，这样他即使拿走了黄帝的神剑他也不算欠黄帝的， 否则旧债才了又添新债，然后再让黄帝他们去过自己设下的五关以取走黄帝的神剑。

黄帝他们往前走了一段路，就看到自己第二位师父，也就是那位打铁造剑的白胡子师父，他手中拿着那把已经给了拦路道人的神剑站在前面等他，黄帝连忙带着黑者上前施礼拜见，师父把神剑交还黄帝说在取出神剑交给那位道的时候我使了一些法术用一把假的把真的换回来了，但对于他们来说那把假的也是真的一样。

黄帝辞别了师父后就带着黑者骑着麒麟飞起来，可是飞了很长时间却好像还在山里，他们正觉得奇怪的时候，只见这只麒麟就慢慢下降，当下到地上的时候它就变成了一匹马。这种天麒麟它是有灵性的，它自己会根据不同的空间不同环境变化不同形象。这匹马停下来也不往前走，就是不断地对着前面喷气，于是黑者上前从腰中取出一个爆炸物往前面空间扔过去，只见那个爆炸物一下炸开但之后又慢慢恢复原样。黑者说前面不能走，有高人设下了一个空间。于是黑者不断用他的爆炸物在前面探路，黄帝就骑着马在后面跟着，七弯八拐的就从里面走了出来，回头一看，那座山没有了，他们是在一条通往神龙氏都城的大道上。

原来黄帝他们去到的那座山其实是在南瞻部洲地上面。
