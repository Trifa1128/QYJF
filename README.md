# 七印解封（16）：悬崖边的西方

2020/09/08
人类文化有东西方之分，而在佛教中又有佛来自西方的说法，这又是怎么回事？修炼人从七印书卷看到，埃及之主是属于佛那个体系的，整个阿拉伯民族对应的也是佛那个体系，印度也属于佛那个体系，耶和华所在的西方神的体系最根本上也属于这个体系，而中国却是属于道家那个体系的，就这两个体系，对应着地表上人类的概念，所以就有了西方佛东方道这个说法，是天上佛道两大体系对应着地上来说的，不是说天上分东西两个方位，于是印度佛教也就说自己是西方的，《西游记》也说唐增到西天取经了。

这种对应不仅仅体现在这些地上各民族的主对应天上的来源，而且也体现在人类社会的管理上。我们前面说过人类社会是受玉皇大帝所在层次直接管理的，但这并不表示玉皇大帝想怎么管就怎么管，他那里也有一个“天意”，他要根据他所在层次展现出来的“天意”去对人类进行管理。在他那个世界里有一套用来接受“天意”的系统，具体存在方式我们这里就不说了，那么是谁给了玉皇大帝这些“天意”呢？我们从七印书卷看到，在天上某个层次中存在着两个特别的天地，这两个天地里都有一座宫殿，殿里都有一个高高在上的宝座，其中一个宝座金色，扶手和宝座背上有许多如意之宝，坐在这个宝座中的神他带有佛的形象，他代表着本次人类文明时期西方社会的“天意”，所以就称为“西天”。另一个天地那个宝座和“西天”那个宝座外形很象，也都是金色的，不同的是这个宝座两边扶手是两条龙，龙身沿着宝座一直到顶两个龙头就从那里伸出来。坐在这个宝座上的神他带有道的形象，他代表着人类东方社会的天意。玉皇大帝对人类社会的管理，大部分都是来自于这两位神传达下来的。所以从人类宗教这个概念来说，欧洲、美洲、非洲、阿拉伯等都属于西方社会。我们在之前说过的被安排来作为耶和华体系对立存在的阿拉伯体系，他们根本上是为了符合相生相克而在一个家庭内部安排出来的事情，是一家子里的纷争。

约在公元前200年前，西方世界的神在与保护外星人进入地球的魔军对战中未能获胜。因为这场战争的发生是由被“西天”管着的那些地上人类所导致的，也就是佛体系的世界里，所以我们从七印书卷就看到，东方道作为一个整体来说不能参与进来。这场对战对于整个宇宙的影响实在太大了，为人类得救的神没能获胜而导致的后果也是极其严重的。我们从七印书卷看到，在本次人类文明开始的时候，代表西方天意的那个宝座上坐着的神展现出来的是一位全身散发着金色光芒的佛，那么在那个时候西方社会体现出来就是正神主导着人类的发展，人类社会处在一个与神相联系的那么一种形式上。而当正神与保护外星人进入地球的魔军战败后，按照正邪双方在战前所定的约，这位神不得不离开了这个宝座，然后由阻挡人类得救的一方指派了一位神坐在那个宝座上，这位神一会儿身体显现的是佛的形象，一会儿又显现为魔的形象，那么在西方社会体现出来的就是人选择神还是选择魔由西方社会地上人他自己的言行来决定，不再由正神主导。于是，对应到西方社会的直接反映就是宙斯退出了西方社会的历史舞台，正神开创的希腊神传文化也随之终结，希腊帝国随即解体，给西方人类消业的罗马帝国随后兴起，外星人的科学与思想也在战乱中正式在西方社会铺开。

外星人进入地球带来它们的所谓现代科学对于人来说可能会觉得有了外星人的技术会给生活带来很多方便，带来更美好的生活，但对于人的得救来说那就太糟糕了。因为外星人的科技是让人脱离神、远离神的。没有正神的指引和帮助，人类就没有希望，其未来就是彻底的毁灭。另外当历史走到耶稣下世时，此时天上还发生了一件极大之事，那就是米迦勒和大红龙的大战也正式开始了。这场大战对于米迦勒所在层次的众神来说不知道最后谁会获胜，但对于耶和华他们来说那个胜败都是明确的，那就是大红龙和它的众多使者一定会被打到地上，旧势力为人得救制造的那个巨关巨难即将要在地上形成了。

耶稣的下世就处在这样一个关键的时刻，西方社会来到了悬崖边！

# 七印解封（17）：犹太人的弥赛亚

2020/09/09
就如犹太先知预言那样耶稣作为犹太人的弥赛亚来了，他降生在马厩，我们从七印书卷看到这是真实的故事。当耶稣降生后，他被放在堆起来的用来喂马的马草上，不是马槽，这一天距今2021年。当耶稣出生时，附近有一户犹太人家，他家房子有一个很大的门，门框上面有一个木板装饰的横眉，就象中国古代门楼门框上用来放对联横批的那样，这时候那个木板突然就掉了下来，结果发现原来木板后还有一块石板，石板上写着一行字，意思就是“弥赛亚来了”。这件事情很奇怪很快就传开了，但相信的犹太人不多，其情形就如今天人们看到贵州藏字石差不多。而另外在离以色列很远的地方有十三位有能力的人，按今天的话来说他们是当时中东那一带的世外高人，圣经故事把他们称为东方的博士，他们看到了犹太人的弥赛亚降生人间了，于是就前来拜见还是婴儿的耶稣，当时只有其中三人能进去看望耶稣，并送上了最珍贵的礼物，其他十人只能在外等候，尊称耶稣为犹太人的救主。于是犹太人的弥赛亚来了这件事情就传开，传到了当时大希律王那里。大希律王就想害耶稣，耶稣的父母只好带着耶稣远走他乡。

七印书卷看到，耶稣他十一岁天目就打开，十二岁的时候与天目有关的其它副通道也打开了，从这个时候开始，就有各种神前来教他各种知识，然后当耶稣提高到一定层次就又有更高的神显现出来教他更高层次上的法，耶稣就是这样与在他下世之前就安排好作为他人间师父的神学习和修炼的。当他差不多30岁的时候，他就达到他来世间需要达到的层次，于是他就着手出来完成他来人间的使命。

对于人间与西方有关的很多事情，坐在西天宝座上的那位时而佛时而魔的神他的身体会产生各种想法，这些想法层层传导下来最后就会影响到人间的很多事情。当耶稣要开始按照当初在天上定下的路去完成他来到世间的使命时，我们从七印书卷就看到，当那位坐在西天的神显示为魔的形象时，它的身体就出现“耶稣要死”这样的一个意思，当它显示为佛时，它身体就出现“耶稣要生”这样的意思，那么他的这个时而要耶稣死时而要耶稣生的天意传到地上，就严重地干扰了当初耶和华给耶稣安排的路。

耶和华在上一次人类文明时期就拣选了犹太人，然后在本次人类文明开始就安排犹太人为耶稣的到来预备道路，就包括我们在前面讲的摩西带犹太人出埃及。拣选犹太人做这件事情是为了达到一个很重要的目地，就是为在末后帮助世人得救传下创世主的福音。这个创世主的福音是什么？为什么要通过犹太人的历史来传达？在基督教故事里有原罪之说，《圣经》里有关于创世和神造人的故事，还有说因为亚当夏娃偷吃禁果而使人有原罪。神造人这个故事是真实的，但亚当夏娃偷吃禁果导致原罪这个事情我们从七印书卷没看到。我们先不管原罪还是自己生生世世干了很多坏事而犯下很多罪，就是说人是有罪的，按佛家说法就是“业”。就象犹太人因为在上一个文明时期欠了埃及人的，所以在本次人类文明时期他们就要被埃及人奴役。摩西把犹太人带出埃及，但他没有能力把犹太人上一个文明时期欠别人的罪给消掉。也就是说，作为犹太人来说，他们在上一次文明时期欠下的对于本次人类文明时期的他们来说就相当于是他们先天带来的罪，然后在本次人类文明时期又会造下新罪。旧罪新罪加一起那怎么还？越往后人类积的罪就越大，如果大到都还不了的时候那只能毁灭。耶和华挑选犹太人就是为了想向人传达这样一个故事，人是有罪的，这个罪靠人自己解决不了，靠摩西也解决不了，只有救主来了才能旧罪新罪一起解决，那么当人没有罪的时候神的国度就真的降临人间了。那么耶和华选定犹太人来做这个事情的时候，他就和犹太人在上一次人类文明时期欠的那些债主签了一个约，这个约其中有一个内容大概是说当耶稣来的时候，如果地上的那些犹太人信靠耶稣是他们的救主，那么这些犹太人就算得救，他们之前欠下的罪就可以勾销，否则他们就要一直承担下去。于是耶和华通过这个犹太人和耶稣演绎的故事，就希望能够给世人传达这样一个信息，也就是人要得救就要象犹太人等待耶稣一样等待创世主到来的时候信靠创世主才能在末后巨关巨难中得救。

可是，那些阻碍人类得救的旧势力他们不这样想，他们针对耶和华的安排实施了精细的破坏计划，首先就是文化断代：我们在前面章节中说到巴比伦来灭犹太人的国的时候，那些搞破坏的就倾尽全力去修改之前和耶和华定下的约，把摩西等先知给犹太人留下的神迹和文化全部毁坏了，那么等到犹太人修建第二圣殿时，那些故事和文化的精华都已经没有了，此时出来主持犹太教活动的也都已经不知道之前摩西留下的那些故事了，就象中国的寺庙经过文化大革命之后，重建的已经不是原来那个了，新庙里的和尚也不是之前那个庙里的和尚了；然后就是引入外邦人，让许多其它民族的人和犹太人一起生活，更甚的是把人种搞乱，使得有些犹太人在与天上的对应上出现混乱，不再只是属于耶和华了，他可能还属于其它体系的主；最后就是派魔转生为犹太人，我们从七印书卷看到，当时那个大希律王它就是魔转生，这个魔在另外空间显现的形象很吓人，嘴巴大的都到耳根了。

这里我们稍微补充一下就是耶和华为什么希望犹太人保持他这个民族的单一，不要和其它民族混合，原因并非种族歧视，而是因为受到了耶稣要做的事情的限制。因为耶和华对耶稣的安排只能是作为犹太人的救主，超出这个范围就超出了当初耶和华与犹太人债主签约的范围了。按照当时的约定，只有犹太人在上一次文明时期的债主才会在犹太人信靠耶稣时赦免那个犹太人欠下他们民族的债，但如果那个人不是犹太人或者犹太人在本次人类文明时期又欠了其它民族的债，那都不在这个约范围内。不在约的范围内，就会带来很大的麻烦。

# 七印解封（18）：依善而行

2020/09/10
圣经故事中讲施洗者约翰和耶稣是非常有缘的人，说他是为耶稣预备世人的，我们从七印书卷看到确实是这样，施洗者约翰在天上是耶稣的使者，他也是西方神的样子，头上也有智慧光圈环绕，身上穿着白色衣服中有一条蓝色绶带，他是右手立掌左手横在胸前跟在耶稣身后走下天门的，然后他以先知的身份为耶稣开路。当耶稣要开始完成他来世间的使命时，约翰已经看到了耶稣将于某个时间来到约旦河，于是他就来到约旦河给人洗礼，等着耶稣的到来。当耶稣达到了他来世间需要达到的层次后，他已经具足了东方佛教中讲的“佛”的智慧和能力了，他知道他在天上的使者施洗者约翰正在等着他，要按犹太先知预言那样给他施洗。于是他就离开家人步行了约四天来到约翰等他的地方接受约翰的洗礼。

耶稣离开约翰后就往城里去，这一天来到一个集市，集市附近有一位独居的犹太女人，集市上很多人都认识她，都知道她患了重病。这个女人正急冲冲地穿过集市，她身上的病又发作了，她非常痛苦，当她经过耶稣身边时，耶稣对她说“你过来，我把你的病治好。”这个女人非常吃惊，他不知道耶稣这个陌生人为什么知道她现在病得很难受。于是她就停下来走到耶稣面前，耶稣伸出右手靠近她胸前停留了几秒钟的样子，然后做了一个象把什么东西从她身体抓出来的动作，就对她说你的病好了，那个女人也发现自己身体不痛了，她相信耶稣真的把她的病治好了，她非常感激，她就问耶稣是谁，耶稣就说“我是你们的弥赛亚”，那个犹太女人马上相信了。这个犹太女人是我们从七印书卷看到的耶稣的第一个门徒，因为她相信耶稣是救主，于是耶稣就为这个女人解开系在她身上的那些债，按约定，因为她是犹太人，所以属于她的上一个文明时期的那些罪的债主们就依约免去了她的罪。当时旁边也有几个路过的犹太人也看到了耶稣治病的事情，他们也相信耶稣就是“犹太人的弥赛亚”，于是耶稣也同样解开系在他们身上的债。



但是在以色列那个地方，不仅有犹太人还有很多属于其它民族的人。这一天耶稣看到路边一棵树下有一位阿拉伯人，他非常饿，饿得都想啃树皮。他身旁有块石头，耶稣用手一指那块石头，然后对那个阿拉伯人说你看那里有一块面包。那个阿拉伯人一扭头果然看见旁边有一块面包，他赶紧拿着面包啃起来，一边啃一边问耶稣你是谁，耶稣说我是犹太人的弥赛亚，这个阿拉伯人相信耶稣的话。这是我们从七印书卷看到的属于耶稣的第一个非犹太人门徒，因为他相信耶稣是救主，于是耶稣也为这位阿拉伯人解开了系在他身上的那些债，但这位阿拉伯的那些债主都不属于耶和华当初定的约里面，于是这个阿拉伯人的所有债都系在了耶稣身上，由耶稣本身去化解。这位阿拉伯人吃完面包就对耶稣说您在这里等我一会，我回去把我家人叫来，我也想他们能够看到犹太人的弥赛亚。那个阿拉伯人就跑回家给他家人说自己见到了犹太人的弥赛亚，他们全家都相信，于是都跟着来见耶稣，耶稣给他们准备了面包，也象那位阿拉伯人一样给他们全家解开系在他们身上的那些债。

耶稣这一路走来，有越来越多的犹太人相信他，但也有越来越多的非犹太人也相信他，于是那些非犹太人转嫁到他身上的债就越来越多。我们从七印书卷看到，开始很多债被耶稣装进了当初在天门那里有一位道送给他的那个球形的法器，后来这个法器装不下了，然后耶稣又拿出其他神给他送的法器装这些债，后来所有法器都用完了，于是后面这些债就象一座座的山一样压到耶稣身上。

耶稣是安排来作为犹太人的弥赛亚的，所以他来到了犹太人建国之地，但耶稣又是依善行走的，只要相信他的人他都会救，而不管那个人的身份地位出生种族，这是他自己的法要求的。圣经故事说耶稣从出来传道到被钉十字架大概三年，大家知道释迦牟尼在印度传法49年，为什么差距会这么大？这里有多种原因导致，其中一个非常主要的原因就是耶稣背负了很多不同民族不同体系的人的债，而释迦牟尼没有背负其它民族的债，他面对的基本上都是他自己体系里的人。所以三年传道，耶稣背负的债就多到已经到他能够承受的极限了，必须要有一个了断。

怎么了断？很多债主就来了，那个时而魔时而佛发出的“耶稣要死、耶稣要生”的天意也要兑现了。

# 七印解封（19）：“大骗子”

2020/09/11
圣经故事说耶稣回到故乡拿撒勒去想要救他们，但却被故乡人耻笑甚至陷害，这是真实的。读圣经关于耶稣的故事很容易让人以为除了拿撒勒，以色列的其它地方还是欢迎耶稣的，只要耶稣一出现然后人们就会看到神迹然后成群的人就跟随耶稣，但事实是耶稣在整个以色列地区也比拿撒勒好不了多少，都差不多。

拿撒勒的耶稣怎么就成了犹太人等待上千年的弥赛亚？他们无法相信。耶稣只在信他的少部分人中行神迹，绝大部分人包括很多信徒都没亲眼看到过耶稣行神迹。修炼人从七印书卷看到，耶稣从他出来传道到被钉十字架共救了约5万人，其中43%为非犹太人，而整个以色列地区约75%的人认为耶稣是“大骗子”！这不是形容词，七印书卷给修炼人看到在这些人思想中直接显示的就是“大骗子”这个意思！

在认为耶稣是“大骗子”的人中，越是犹太教里身份越高的认为是“大骗子”的越多。这不是说这些犹太教徒表面上是犹太教徒背地里不信耶和华不信摩西，他们背地里同样认耶和华是主认摩西，但他们就是不认耶稣。这不能完全算是犹太人的错，因为破坏人得救的一方在巴比伦灭亡以色列时毁灭了犹太人的历史，这是其中一个原因；还有一个就是魔转世，当时的犹太领袖从七印书卷看是一个由紫色火组成的人形的生命，这些生命在后面兴风作浪。

耶稣出来传道时已经具足了洞察世间一切的能力，他知道救非犹太人面临的后果，特别是救阿拉伯人面临的后果，因为我们前面也说过，阿拉伯这个体系是被安排来和耶和华体系对立的，但既然是弥赛亚就不能挑人，只要那个人遇到他信他，那么他就必须救，必须为他解开系在身上的罪。这一路走来，当耶稣骑着驴子走进耶路撒冷时，我们从七印书卷看到的画像和西方宗教关于耶稣进耶路撒冷的画就很不一样了。在宗教故事中，说当耶稣来到耶路撒冷时，许多人听到他来了，就去见他。耶稣骑着一头驴驹子进城，曾有先知预言，说神的儿子会这样做，许多人相信耶稣就是神的儿子，他们把棕榈树枝和衣服铺在地上，让他骑着驴子过去，并高喊耶稣是他们的王。法利赛人很害怕，认为每个人会因此相信耶稣，担心以后没有人会听他们的话，于是设计陷害耶稣。这个故事听起来是耶路撒冷的人们夹道迎接救主，然后只是一小群的法利赛人想着谋害耶稣，但我们从七印书卷看到了完全不同的画面。



从七印书卷看到，耶稣确实是骑着驴子进城的。当他进城时，有一些相信他的人跪在路上迎接他，有些拿着棕榈树枝，但数量不是很多，而在耶稣身后以及周围是一大群人来看笑话的，七印书卷给我们展示了这群人当时的思想：

----为什么要骑头驴子进来？

----为什么不骑个金马？

----为什么不能象神一样飞进来？

----为什么不能有人抬他进来？抬进来才对啊！

----或者骑个野兽进来不是更好看吗？或者飞马啊！

----我的天啊怎么是头驴子？

----。。。。。。

从人的角度说此时的耶稣是很凄凉的。一方面这些人的一思一念耶稣全知道都看得见，但耶稣慈悲，他不能因为人怎样想而不去救他们；另一方面，人的肉眼只看到耶稣轻轻松松地骑着驴子进城，露出慈悲的笑容，但七印书卷给修炼人展现的是这样一个画面：他的身体正被一座巨大的山一样的黑色物质压着，猛一看就象一只蚂蚁身上扛着一匹黑色大象。这些黑色物质就是那些被耶稣救的人他们的债也就是业力转到耶稣身上来的。不是说耶稣就扛着这座山一样的黑色物质就完了，七印书卷给修炼人展现出来这些黑色物质实质上是各种痛苦它们不断地往耶稣身上施加：

----全身被火烧

----非常饿但吃不了东西

----几乎无法呼吸

----全身到处都在痛

----身体被动物不断地撕咬

----被刀一片片切

----。。。。。。

耶稣知道，天上众神也知道，耶稣必须在耶路撒冷结束他在人间的使命了断这一切回到天上，否则不断增加的债很可能会让他毁在人中。于是在耶稣走进耶路撒冷的同时，天上正邪双方以及那些耶稣背负的债主们在一起也开始了一场关于耶稣需要怎么了断这一切才会让大家都接受而返回天国的会议。我们只能用“会议”来说了，因为从画面上看他们是坐在一起。

# 七印解封（20）：十字架的恶报

2020/09/12
耶稣要想回到天上，他不能把这些债也带着，他必须在人间把这些债还干净了才能回去。欠债要还，耶稣身上背负的那些债对应的那些主，他们有绝对的权柄处置这些债，而耶稣只能按照他们定下的来还，就象前面讲摩西带犹太人出埃及，那条路不是由耶和华定的，而是那些债主定下的。从七印书卷看到，天上债主分两派，一派叫善解，一派叫恶报，于是就有了两种还债的方案：善报方案计划让耶稣把那些罪归还一部分到那个被耶稣救的人身上，由这个人在将来修炼圆满后福报于他们的世界，然后耶稣可以按照尸解的方式善解剩下的债而从人中解脱出来；恶报方案则计划让耶稣用他的身体承担所有这些债对应的痛苦，耶稣以被钉十字架的方式从人中解脱出来。两派互不相让，最后大家形成了一个意见，就是想善解可以按照善解的方式处理属于他们的债，想恶报的也可以按照他们恶报的方式处理属于他们的债，但耶稣是以尸解还是以被钉十字架的方式从人中解脱则交由地上犹太人来决定，因为耶稣是以这些犹太人弥赛亚的身份来的，这就是西天宝座上那个时而佛时而魔的天意表现。

新约里很多故事都是真的，但在细节上或者语义上或者环境上和事实有出入，例如耶稣骑驴子进耶路撒冷受到人们欢迎，这个受欢迎的程度就有出入：是有人欢迎，但不是夹道欢迎。又例如说“救主会再来”这是真的，但圣经故事把这个语义写成耶稣会第二次来那就错了。

圣经故事说耶稣在客西马尼园受苦，祷告的时候开始因疼痛而颤抖，他非常痛苦，血像汗珠一样滴下来，这是真的，但里面又说耶稣在客西马尼园不想受苦而向耶和华祈祷，我们看到不是这样，不是耶稣不想受苦，他作为犹太人的救主，受苦是他来人间前就已经知道是必须的。我们看到耶稣他是因为不愿意以被钉十字架的方式解脱而向耶和华祈祷。虽然说最后是否会被钉十字架由地上的人来选择，但耶稣和耶和华都知道，地上犹太人已经被魔掌控了，钉十字架就变成了必然，所以耶稣向耶和华祈祷希望能够换一种解脱的方式，但耶和华也没办法，这不是他能决定的。

于是故事依次上演，耶稣被犹太人逮捕交给罗马行省长官彼拉多代表罗马人裁决耶稣的命运。罗马是属于耶和华的，当时罗马帝国还处在强盛时期，也就是说总体上还在耶和华的掌控之内。耶和华说耶稣不能被处死，于是彼拉多就说耶稣没有错，耶稣应该被释放。犹太人不同意，耶和华就借彼拉多的嘴警告犹太人，如果你们要耶稣死，那么你们就要为他的死负责。当犹太人说他们会负责，天上定下的那个恶报的解脱方式就开始执行了。于是地上耶稣被戴上十字架，那些善解的债主就一一上前从耶稣身上取走一部分债，由那个被耶稣救的人在未来修炼圆满时福报，然后留下一部分让耶稣肉体在痛苦中归还。人看耶稣就背着个十字架在路上走向刑场，其实耶稣每走一步都非常痛苦，就象被无数的针扎在脚上。

走到刑场，善报的罪都还完了，然后耶稣的肉体被钉在十字架上开始还恶报的债。人觉得被钉子钉进四肢那肯定痛不欲生了，但其实那个痛对耶稣来说不算什么，真正痛苦的是承受那些恶报债主带来的痛。我们从七印书卷看到，那些恶报债主们商定了一长串的恶报名单，然后写在一个书卷上，交给一位神，这位神的头上带着老虎头一样的头盔，手中拿着一根狼牙棒，带着长长的一队执行恶报的神出现在耶稣被钉十字架的上空。

只见那位带老虎头的执行掌官高喝一声，然后就从行报队伍中出来四位神，他们全部将士打扮，其中一位银色盔甲，两位金色盔甲，一位铜色盔甲。四位中有三位戴古代将军的头盔，一位戴古代蒙古族那样的帽子。这四位行报神手上都拿着弓，弓的两头象黑色的鸭子头，它有一个名堂叫亿箭齐发万箭穿心。只见这四位神把弓拉满，弓上突然就出现无数的带红色羽毛的箭，亿箭齐发全部射到耶稣身上，把耶稣亿箭穿透万箭穿心痛不欲生。

当第一批神执行完，那位带老虎头的执行掌官又高喝一声，然后第二批行报的神就走出来，共有五位，全部将士打扮，其中两位穿黑色盔甲，一位白色盔甲，一位银色盔甲，一位红色盔甲。五位中有一位拿银色刀，那刀不是普通的刀，刀护手左右两边各突出三个尖刺，这把刀只要一落到耶稣身上，耶稣就会觉得自己的心脏被他自己不断地撕咬；又有两位也拿着银色刀，但刀的护手上没有刺，他们的刀只要一砍到耶稣身上，耶稣就象同时得了几百个癌症一样痛苦；还有一位拿着赵子龙那样的枪，那把枪只要一刺到耶稣身上，耶稣就要把自己的心脏吐出来；还有一位拿铜色锤子，锤子的柄象枪那么长，这把锤子只要打到耶稣身上，耶稣的思想中就会想到各种各样非常不好的东西。只见这五位神同时举起各自手中兵器一起击向耶稣，个中滋味读者自己体会了。

第二批执行完第三批上来三位西方神，只见他们左手掌心向上横在胸前，右手掌心对着耶稣发出强大的光，这个光比孙悟空在太上老君炼丹炉里的三味真火还热，个中滋味估计只有孙悟空能想象一二了。

负责执行最后一个报应是一位穿木色裤子赤裸着上身的神，他的拳头快的就象无数的闪电一样击打到耶稣身上。拳头所过之处，耶稣身体的精华就象被抽干了一样，血肉在快速消失，就剩骨头一般。这位神拳头一收，耶稣大喊一声，他身上背负的所有债就此了结，人间肉体也在痛苦中死去，耶稣就这样从人中解脱出来了。

修炼人从七印书卷看到，耶稣在十字架上共接受了139个批次的恶报。

救人不是随口一句话，要救世中人绝非神一样的来神一样的去，耶稣只是救了约5万人就要为那些被救的承受如此巨大的痛苦，试问末后世间创世主为救众生到底要遭多大的罪？
