# 明见地狱见闻（1）：地狱分布

2020/10/26
玉皇大帝所在的天地投影到三维空间来看，它的外形是天圆地方那样，地由四个海组成，他们是东海、南海、西海、北海。东海里有东胜神州，南海里有南瞻部洲，西海里有西来贺州，北海里有北俱泸州。

在用来划分四海的那个纵横分界线的交汇处有一个象巨大的火山口那样的存在，是这一层次魔王居住之地，那里有一个冥府，冥府里有地狱，是整个四海里最大的。在这个地狱的下面，有一个无生之门，被投入无生之门的生命将在永生永刑的无尽痛苦中被灭尽。

四海每一个海都有一个冥府，里面都有地狱，统管该部洲所有冥府。

南瞻部洲里有一个海，它对应到人类生活的这个地球，为了行文方便，我们称其为地球海，上面有十九块地，其中一块地在海的中央，另外十八块地星罗棋布分布在中间这块地的周围，整个地球所有的国家都在这十九块地上，而中国和蒙古在中央那块地上。这些地它其实是浮在这些海上的，下面都有生命在稳定着这十九块地在海中的位置而不至于飘来飘去。与地球对应的冥府共有二十个，其中地球海十九块地上都有一个冥府，在中国那块地下面海底那里，也就是地球海的海底中央那个位置上有一个冥府，它统管着地球十九块地上的冥府。这些冥府里的地狱大小不一，地球海中央那个地狱最大，然后是中国那块地的地狱。蒙古和中国在同一块地上，所以蒙古人死后其实也会被押到中国那个冥府里。美国和加拿大是在同一个冥府范围内，日本、朝鲜、韩国在同一个冥府范围内，澳大利亚、新西兰也在同一个冥府范围内。

人类社会有刑法等各种法律条文，上面列明了针对不同的犯罪类型和罪行等级应该怎么量刑，冥府那里也有。每一个冥府里都有判官，根据一个人的业力大小以及此人在人间所犯罪行种类而裁量应该执行的刑罚。如果这个人犯的罪是针对修炼人的，那么这个罪就会罪加一等。历史上在耶稣弟子修炼的那个时代，迫害耶稣弟子之人都会罪加一等，释迦牟尼弟子修炼时代迫害他们也会罪加一等，今天是创世主弟子修炼的时代，迫害大法弟子的罪就会倍增。

真修之人如果没修成就死了，那么他的死不归冥府管，由他师父直接管。

地球上的一个人如果死了，如果他生前处处与人为善德很多，他就会先来到一个地方，这个地方不属于冥府，那里有神会问他是想先到地狱把业消了还是用自己的德把业换了，如果他选择先到地狱把业消了，那么他就会被安排到冥府里的地狱消业然后根据他的德到三界不同层次做天人，特别的则会成为三界不同层次的神；如果他选择用德换业那么那里的神就会给他做一个德和业的转换，然后他直接去三界做天人，特别的也会成为三界不同层次的神。而那些业力较大的没有选择的机会，将被直接押到那块地对应的冥府里根据他的罪业接受地狱里的各种刑罚以消业。

地狱里用来消业的刑罚很多，历史上传说有刀山，火海、下油锅等等，这些都是真的。地球海中央那个地狱里的刑罚与地球那十九块地里的地狱刑罚不同，南海地狱里的刑罚与地球海中央那个地狱里的刑罚也不同，四海中央地狱里的刑罚与南海地狱里的刑罚也不同。所有这些刑罚只能过一遍，第二遍就不起作用了，这是法要求的。所以如果一个中国人死了来到中国那个冥府地狱里，如果刑罚过一遍这人的业还消不完，他就会被押到地球海中央那个地狱里，如果刑罚过一遍业还有，就会被押到南海那个地狱里，如果刑罚过一遍业还有，就会被押到四海中央那个地狱里，如果所有这些地狱里的刑罚走一遍业还没消完，这个人就会被投入无生之门在永生永刑的无尽痛苦中全部销毁。

现在人类业力滚滚，被投入无生之门的很多。

# 明见地狱见闻（2）：地藏菩萨已转生

2020/10/27
《西游记》第三回说美猴王喝醉酒睡梦中被两个鬼差索了命带到冥府，美猴王一看大怒，从耳中掏出金箍棒打上森罗殿大闹冥府。孙悟空来到的这个冥府是属于东海的那个。然后《西游记》第十回说唐太宗地府还魂游历十八层地狱，这个地狱和孙悟空的那个不是一个，是属于南海冥府那个地狱。

统管四海每个海地狱的冥府不在那个海里面。四海之间有一个分界线划分，每个海的冥府就在这个分界线的下面。说是分界线，其实是一条非常大、里面都是火的沟，冥府就在这条沟的底下某个位置上，与这个海那块部洲平行。例如，东海那个冥府是在用来划分东海和南海的那个火沟底下，南海的那个冥府在划分南海与西海那个火沟底下。东海冥府和南海冥府都是十殿阎王，他们身材高度普遍比人要矮一个头。

佛教故事中有说地藏菩萨曾在忉利天大会的众生面前，受释迦牟尼的咐嘱，在释迦牟尼圆寂后至末劫弥勒佛来之前在冥界担任导师。我们看到这个故事是真的。在南海冥府所在位置的上面有一个岛，这个岛与南海海面是平行的，它就象浮在火海上的一块石头。这个岛虽然在火上面就象被火烤着一样，但它其实是一个清凉世界。岛上有一座宫殿，瓦是绿色的琉璃玛瑙之类的，被称为翠云宫，地藏菩萨就住在这里。翠云宫外有一个道场，那里有一个带阶梯的通道直通南海冥府，道场用人的时间来算的话大概一周会开一次，然后那个通道就会打开，冥府里的众生和来这里受刑的人这时候就有机会到这里听地藏菩萨讲法。

《西游记》故事说真假孙悟空来到幽冥地府，说地藏菩萨那里有一个灵兽叫谛听，它能听出哪个是真的孙悟空哪个假的。我们看到地藏菩萨那里真有谛听这么一个灵兽，但它长的样子也不是有些雕刻或者画像画的那样，它其实象一只威武的警犬，而与警犬不同的在毛色、眼睛和耳朵。它身上的毛是橙色的，上面有黑色的、象老虎那样的纹路，脸上有五只眼睛，头部有十二只耳朵，其中头顶一对、下颌一对、然后从上到下左右两边各四对。

佛教有些故事说地藏菩萨是冥王的领导，我们没看到这种情况。地藏菩萨的道场只对南瞻部洲那个冥府开放，而且只有地球上的人，如果他在死前真的相信地藏菩萨，那么当这个人死后，一般来说，他身上的业能减少一点点，但这一点点的业可能也会让那个人少受一些罪，另外，这个人如果还没到被投入无生之门的，那么他在地狱经历了种种刑罚后，他就可以避免被安排转生为石头之类的，然后会被安排进入轮回。

关于地藏菩萨的传说中，可能“众生度尽、方证菩提；地狱不空、誓不成佛”这句话最为有名了。但我们看这句话多半不是原话，很可能是被后人歪曲了。地藏菩萨已是菩萨果位，他非常清楚地狱对于四大部洲来说是不可缺的一部分，地狱不可能空，特别是对于南瞻部洲地球里的人类来说，在滚滚红尘之中，在迷中，人很容易就造下罪业，如果没有地狱的存在，地球上早就没人了，几乎早就在无生之门里销毁了。

其实今天的人再怎么信地藏菩萨都没用了，因为地藏菩萨连同谛听在很多年前已经不在翠云宫，翠云宫前那个道场也关了，通向冥府的那个通道也关了。现在翠云宫那个岛上还有一些看守的官员之类的，以及一些动物。佛教故事说释迦牟尼咐嘱地藏菩萨在他圆寂后至末劫弥勒佛来之前在冥界担任导师，今天释迦牟尼说的弥勒佛已经来了，地藏菩萨也在约三十年前转生为男性白人，约在四年前成为一名大法弟子要在这世修圆满证得更高果位。谛听也跟随着转生为狗，样子也象警犬，在他成为大法弟子的前一年，谛听这只狗就跟随了他一直到今天。

# 明见地狱见闻（3）：阴间之苦

2020/10/30
一般来说，一个人到寿死了，如果没资格去神问话的地方，那么他就会瞬间落到阴间，然后就会看到有鬼差用锁链往他脖子一套，链子象人间的铁链子，手上戴着古代犯人木板那样的铐，就被拉着上路了。鬼差一般两个，象中国人说的黑白无常，每次出来办公，一般会从人间带走六七个人，锁链上串着。

即使是全世界权力最大的什么党魁、或者获得多少个诺贝尔奖的科学家、或者是全球首富、或者是世人追捧的巨星，等等，所有这些权力、名头、财富、追捧，被鬼差拿锁链往脖子上一套，就什么都不是，就是阴间里的一名犯人，言行上稍不对路，就会招来鬼差一顿谩骂，然后是一顿皮鞭抽，或者棍子打，这是真实的。反而那个一生吃尽苦头的心地善良的乞丐，他可能去到神问话的地方了。所以人世间越是地位高其实越危险，一旦落到阴间就越惨。

阴间和人间是非常不同的世界。一个生命转生为人他的记忆会被抹去，但一个人死了，他在人世间所形成的所有记忆，就算是想过的一念、做过的鸡毛蒜皮的事情，全部都会牢牢地在这个人的大脑里被带到阴间。带着这么一个人间形成的记忆看到阴间的景象，所有人都会不想活但又死不了，都想回到人间但又没任何办法能离开。

阴间的空间是灰蒙蒙的，这里也生活着很多生命，大部分长着就象中国故事里描写的那些鬼，也有这个空间的山水树林和动物，有些树会长出粮食，所以阴间的生命其实是不用种地的，就直接吃树上长出的粮食，他们也会吃这个空间里的一些动物的肉，也有房子住。对于这里的生命来说他们也有他们喜欢吃的食物和不喜欢吃的食物，但对于死后带着人间所有记忆来到这里的人来说，这一切都是粪便组成的一般，看着都恶心，到处都象粪便一样脏，连空气都象那样。这种感觉，应该和人间的人被沉到粪坑里的感觉估计差不多，苦不堪言。

人死了要被黑白无常用锁链套住，我们看其中一个原因是如果不被锁链套住，那个人忽然发现自己死了来到的居然是这么一个地方就会象发疯一般嚎啕大哭四处狂奔了。所以人死了落到阴间，那真是非常悲惨的。这种惨，是因为人间相对于阴间来说就象天堂，他带着天堂般的记忆来到了粪坑里产生的巨大反差造成的。为什么生命转生人间要被抹去记忆？为什么天上的神都害怕犯天条被打下人间？这是其中一种原因。

当然，过了一段日子，被带到阴间的这个人也不得不习惯阴间这一切了，那些早来阴间的那些人也会告诉这个新来的很多关于阴间的知识，这个人自己也在慢慢学，鬼差也会一路的讲这个世界的规矩等等。这时候，所有人都会后悔当初在人世间为什么做下那些坏事，都后悔为什么不信神为什么不听劝。反正就是悔青了肠子，可是晚了，苦不堪言。

虽然人是从天堂般的人间来到阴间的，但到了阴间的这个人却又是比阴间里的生命还低等，因为来到这个空间的人就算是罪犯，全部都按照阴间罪犯那样对待：被锁链锁住被鬼差呼来喝去，住牢房，吃牢饭。虽然看着阴间那些树长出的食物象粪便，其实来到这里的人连那些食物都很难吃上，牢饭是一种象人间布丁那样的食物。我们看了一下这种食物的制作过程，大概是这样：这里的生命拉出的大便，在上面加一些那个空间的调料，熬成一块块象人间的布丁那样！不吃就很饿，吃又无法言表的恶心，苦不堪言。

中国有句话叫“好死不如赖活”是有它的道理的。人在遭受痛苦各种打击时总想早点死，说什么“一了百了”，其实真的死了就会发现不仅没死了，反而只是更漫长更巨大痛苦的开始，这时候都想赖活了。

丰都鬼城在中国历史上非常著名，有很多关于阴间的传说，也有说就是冥府地狱所在。其实重庆丰都鬼城那个地方不是冥府地狱所在，但它也的确是一个特别之地。在那个地方有一座鬼城，那个巨大的城门上就写着“鬼城”两个字。这座“鬼城”就象一个中转站，地球上中国和蒙古，还有台湾，那里的人死了来到阴间，都会被当成罪犯一样先押到重庆丰都这里，在鬼城里等着去冥府。

可能很多人会以为地狱和阴间是在一块的，其实地狱冥府和阴间不是在同一个空间，所以阴间里的生命其实也不是随便就能去冥府地狱的。重庆丰都鬼城那里有一条通向中国那个冥府所在的通道。这条通道很窄很长，里面有很多阶梯，它就是专门设计成只能容一个人通过那样。中国那个地方的冥府共有六殿阎王六层地狱，它的位置不是固定的，在地球海中央那块地里飘来飘去，有时候在内蒙，有时候在河北，有时候在蒙古。等等，反正就是经常换。可是除了中国，地球海其它地方的冥府是固定的，包括四海那个冥府和四海中央那个冥府，都是固定在那个地方的。为什么只有中国这个冥府的位置飘来飘去我们也不是很清楚背后的原因。但无论冥府怎么移动，重庆丰都鬼城那个通道却总是一直连在冥府那个出口那里。

现在这条通道里面从冥府那个出口那里一直到丰都鬼城入口这里，这个通道里面一个连着一个，阶梯上站满了前去冥府接受审判的人。那些还轮不上进入这个通道的，全部被关在鬼城大大小小的监狱里。因为死的人很多，通道又这么小，所以在丰都鬼城关着等候的人很多，等待的时间遥遥无期。

在人间，有很多关于监狱里的狱警怎么虐待犯人的故事，其实丰都鬼城里看守人的那些鬼差可能更恶，各种克扣各种虐待，反正被关在里面就没有舒服的时候。鬼差也会接受犯人的好处。所以如果来到这里的人身上有钱的话，那么也是很大可能买通鬼差的，然后鬼差就会让这个人在这里过的舒服些，比如可以从这些鬼差那里买一些这里生命吃的东西，例如树上长出来的那些粮食，一些蔬菜之类的，也就是只能这样。这个钱不是人间那种钱，很象中国古代市面上流通的那种中间带孔的钱币。

现在地上人死了亲人一般会买些纸扎的房子、车子、多少万亿的冥币等等烧给他，还有烧iphone等电子设备的，觉得阴间用的上。其实阴间几千年来还是中国古代历史故事描述的那个样，没怎么变，是人类社会变了。烧这些东西没用，过不到那边，只能是人间亲人表达一下孝心，就这样。真正有用的是地面上的亲人为这个去到阴间的亲人破财。人间这边为他破多少财，就会有一定数量的阴间钱币去到这个人身上。然后这个人就可以用这些钱让自己在阴间等待去冥府的这段时间过的舒服些，买些东西吃。一旦这个人进入去冥府的通道，人间的亲人再做这些就没用了，为他破的财也过不到他身上。

中国古代有过世亲人托梦说自己在阴间如何如何，让人间亲人如何如何，等等这种故事。这是可能的，只要那个鬼差愿意帮他做这个事情，而且这个鬼差也有这个本事让那个人托梦，就能够托梦。托梦之类这种事情也只能在阴间这里做，进入去冥府通道后就托不了了。

其实真正能帮到阴间亲人的，就是人间的亲人帮这个阴间的亲人还那个债，比如那个人生前骗人多少钱，这边亲人帮着把这些钱还了，或者这个人生前迫害了谁，人间的亲人帮着把这个结解了。但现在很多人好坏不分，所以即使有这个心，也搞不清楚怎么还了。如果真能帮阴间亲人把这个债还了，那这个人在阴间的时间可能就会短，在地狱也可能不用受那么大罪。

古代故事有说为过世亲人找高僧道人做法事的，说可以免除亲人阴间和地狱之苦。这要看做法事的是什么人。如果是真修很高层次的，是能够起一些作用。可是现在人世间哪里能找到真修的高层次的僧人道士？很难很难很难很难很难了。

阴间这些苦，其实也是人自作孽导致的。

佛教故事有说如果善待修炼人会得福报，会免去很多阴间地狱之苦，这是真实的。但现在时代也变了，现在不是释迦牟尼弟子修炼时代，也不是耶稣弟子修炼时代，现在是创世主弟子修炼时代，世人如果善待大法弟子，是可以免去很多阴间地狱之苦的，甚至都不会去阴间。而如果死的人家里有真修大法的，那么这个人死了，因为他家有人修大法的福份，他一般都不会到阴间。即使一个人死了，他已经在阴间这里，或者已经在地狱那里正在遭受刑罚，如果就在这时候人间他有一个亲人开始真修大法了，就这种情况，这个人一般也会被立即从阴间或者地狱那里提到天上被神问话的地方从新安排他未来的去向。因为现在是一个特别的时代，创世主弟子修炼的时代！

悠悠万世几人不迷？

# 明见地狱见闻（4）：江某某之死

2020/11/02
中国古代有关于元神的说法，也有关于魂魄的说法，说这个元神或者魂魄才是真正的这个人，佛教故事说“臭皮囊”，说这个肉体只是元神或者魂魄穿的一件衣服。如果这个人的元神被黑白无常勾去，这个元神就成了阴间的鬼，人间的这个肉体就断了气死了。我们看基本是这个情况，但也有例外，就是曾经的中共党魁，今天在人世间看着还活着的江某某，它的元神其实在二零零二年的时候就被黑白无常勾去，真正代表江某某的这个人实际上那个时候就死了。

在二零零二年某个时间，很高层次上有一位佛，身上有法轮，在我们所在层次看到这位佛不是创世主，他派了一位使者持法旨直接送到玉皇大帝那里，玉皇大帝接了法旨展开一看，上面写着要在二零零二年九月三日某个时辰把江某某的元神勾到地狱受审。玉皇大帝立即与满殿文武商量。如果夺取世间一个普通人之命，就让冥府派两个黑白无常去就是了，但江某某这个人比较特殊，它被旧势力选上作为“考验”创世主弟子在人间修炼的打手头目，层层空间有很多旧势力的坏神在保护它，并不是玉皇大帝能动了的。但这个法旨来的层次非常之高，玉皇大帝必须办，于是大家就商量，只能尽全力而为了。商量妥当，玉皇大帝就给四海中央的冥府、南海中央的冥府、地球海中央的冥府、中国的冥府，下发了在二零零二年九月三日某个时辰勾去江某某元神到冥府接受审判的玉蝶，让各大冥府按照法轮世界佛的法旨做好准备。

到二零零二年九月三日这一天，由四海中央冥王派出一队冥府将士保护着两个精明的黑白无常到了江某某住的地方，这个地方不在北京城里，而在北京城外，而同时在天上，玉皇大帝亲自带着几乎整个天庭的将士来到上空，与此同时，层层不同层次漫天众佛道神等也来到上空。当时发生了一场小规模战争，就是那些保护江某某的一些坏神出手阻拦，但这次前来帮助取去江某某元神的众佛道神太多了，那些出手阻拦的很快就被众神销毁，那些更多的保护江某某的旧势力坏神一看这漫天众神，都想搞不定了，于是都四散。那时候是晚上，江某某正在睡觉。从它睡觉地方来看，它还是很喜欢中国传统那些东西的，满屋子都古色古香，都是中国古代那些座椅器具，床也是皇帝睡觉那种龙床。

因为那些保护它的势力都四散了，时辰一到，黑无常就上来用手中那条黑锁链一套，江某某的元神就被拘到阴间，这时候人间那个肉体就断气死在那里了。江某某的元神一到阴间，它这时候元神的外形还是和人间那个江某某长的一样，还没被打回原形，但身上的打扮却立即变成这样：穿一条灰色的囚裤，身上是白色囚衣，囚裤和囚衣到处都是洞，也有很多不同颜色的补丁，看着就是一件不知道反复用了多少年的囚服，衣衫褴褛的样子，脖子上有一条绳子下面挂着一块长方形木板，上面写着三个白字：江某某，手上又被一个古代那种木铐子铐着，脖子还有一条黑色链子锁住，链子另一头在黑无常手上。这时候我们看到一个很滑稽的对话。那个江某某一到阴间，它也马上醒来一看自己明明是在睡觉的，难道是做梦？它还没反应过来，它还以为自己是一手遮天的党魁，就恶狠狠地吼道“你们是谁？想干什么！你们知道我是谁吗？”白无常说“那你又知道我们是谁？”江某某就说自己是谁有多大权力，可以立即把全国的军队调来把你们统统毙了。白无常说“喂，看清楚，这是阴间！”这时候江某某才回过神来看周围的情况，发现这个地方和中国历史上传说的阴间怎么那么象，黑白无常也一样，还有那些周围的冥府将士，已在阴间的江某某这时候才切实知道这是真的阴间，不是做梦，当时就吓得瘫倒地上，连哭带拉的不断哀求，说自己有多少钱，有多大权，只要放回到人间，你们要什么都给。只见白无常轮起皮鞭就是一顿猛抽，边抽边数落江某某迫害法轮大法恶贯满盈，今天奉法轮世界佛的法旨拿你去冥府审判。江某某一听法轮大法，当时就被吓得魂飞魄散一般，黑无常一扯链子，瘫痪在地的江某某就立即被扯起来，在冥府将士的保护下飞快地离开了，众神佛道也随即离去。

可是，不知从那里来了一个极高层次的旧势力坏神，佛的形象，说什么大法弟子的修炼还没有结束还必须继续接受“考验”，江某某还要“活着”，它就用法力把那些四散的保护江某某的旧势力坏神聚集起来，然后带着它们立即返回江某某尸体那里，使了一些手段，就让那个尸体“活”了过来，在肉眼看来它还是原来那个江某某，但实际上它就是一个空壳那样，里面没有元神，或者说里面没有它的魂魄，有些象聊斋里说的画皮。

我们看到，从四海中央冥府出来执行公务的这两位黑白无常并没有按常规把江某某留在重庆丰都鬼城那里等候，而是立即以最快的速度把它直接押到中国冥府那里接受审判。

与江某某相比，我们知道去年中国有一位前总理去世，他的死就不象拘江某某元神那样大费周章，他就是到了生死簿上写明的死期了，然后中国冥王派黑白无常正常办公，在写明的时辰把他的元神拘到阴间，也是衣衫褴偻的囚服，手戴铐脖子挂着一个木牌，上面用白字写着他的名。他生前虽然位高权重家财万贯，可一到阴间就是一身破破烂烂的囚服身无分文，什么都带不走不仅在人间白忙活一场，还造下满身罪业。他的家人为他请了很多法师搞了一场很大的法事，我们看到也是白忙一场，一个钱子也去不到他身上，想超度那更绝无可能了。他被带到重庆丰都鬼城后，也象普通犯人一样被关在监狱里等着，这一等大概就是两年才被押到中国冥府那里受审。阴间的时间比人间要快很多，人间一年那里已经过去好多年了。但对于身在阴间的他来说，被关在鬼城监狱的两年那都是实实实在在的，一毫秒都不会错过。因为身上一个钱也没有，他在监狱里受尽虐待，也只能吃用鬼粪便做成的布丁那样的牢饭，这一吃就吃了两年。

看见阴间很苦，可是和地狱相比，阴间之苦就算不上什么了！

# 明见地狱见闻（5）：江某某鬼灵地狱受审

2020/11/03
从重庆丰都鬼城进入中国冥府的入口上面刻着三个黑色大字：无空洞，洞四面都是铜墙铁壁无法穿出去。洞很长，但里面空间不大，刚好能容鬼差押着罪犯一个个进去，进入后步子就只能往前迈不能后退，有去无回。所以人们说人鬼殊途，那真是这样，一到阴间，人间的家人、事业等等一切都和他没有关系了。从无空洞那头出来，就会看到前面是一个巨大的四方形棕色通道。整个通道里什么都没有，阴阴森森，尽头有两扇门，门差不多有通道那么高，门洞大小就象一个小人站在天安门门洞那样，洞门上方门眉那里刻着三个白色大字：鬼灵市。

如果在稍微高一些境界来看，就会看到其实鬼灵市所在空间的外形象一座山，里面是一个小世界，为了便于大家了解，我们画了一个示意图。这些在过去都是属于不允许人知道的真相，即使是在释迦牟尼弟子修炼时代，还是基督教、道教等各门各派，对整个四大部洲以及冥界的描述只能东一块西一块地说，都是不允许象我们今天这样系统揭示的。但因为今天是一个特殊的历史时期，大法弟子修炼的时代，我们也就能够把它写出来。不仅仅鬼灵市，我们后面还会把整个地球海、南海、四海中央的冥府以及无生之门的存在和其中刑罚都详细描述出来。其实我们“明见神话”其它几个系列也是这样，揭示了很多过去都不允许说的真相，今天能够写出来也实属不易。

人世间一个人犯了罪有监狱等司法系统等着他，其实从某种意义来说，阴间、冥府、地狱都是为人而存在为人而造的，它们的存在就是为人类专门打造出来的司法系统。人世间的司法系统针对的是人在世间犯下的某项罪行进行审判和惩罚，而阴间、冥府、地狱针对的是人在世间一生所做下的所有事情的审判和惩罚。所以人死了如果去不了天堂，他下到阴间就百分百是地狱里的罪犯。中国、蒙古、台湾人死了来到阴间，都会被押到鬼灵市受审，而整个鬼灵市是服务于鬼灵地狱那个地方的。

鬼灵市有金河和红河，金河河水是金色的，红河河水是红色的，开始分别从四个方位流入，然后合并一起流出。金河包住的那块蓝色区域就是在地狱里办公的那些冥府将士、官吏、鬼差们生活的地方，那里有很多中国古代那样的房子，就象小市镇那样，共有四块，外形和位置就和我们画出来的样子差不多。这里也是一个有情世界，他们在这里也有他们的家人。那些空白地方都是森林，也有田地之类的，他们在上面耕种。鬼灵市有六座阎王殿，分布位置就大概在我们示意图画出的那样，里面是六殿阎王办公和居住的地方。

鬼灵地狱外面是高大的城墙，就象中国古代城墙那种，上面有冥府将士巡逻。围绕着城墙，外面又立着一些宝塔，镇守着鬼灵地狱。罪犯从重庆丰都鬼城那里走无空洞被押到鬼灵地狱，这些罪犯都看不到鬼灵地狱外面的风光，都是在一个封闭的环境里走完全程。鬼灵地狱之大对于在其中的罪犯来说就象无边无际一样。进到鬼灵地狱，那里有一个收押待审罪犯的监狱，附近有一座衙门，就是专门用来审判罪犯的。当罪犯被地狱判官根据其在人间所犯罪业判完后，就会被押到行刑入口那里，那里也有监狱，是用来收押审结后等待发到地狱受刑的罪犯的。鬼灵地狱共有六层，从行刑入口那里进去，层层往下，一直到六层，每一层都有收押罪犯的监狱。当六层地狱的刑罚都走完，这个人的罪业还没有被执行完，就会被发配到地球海中央那个冥府等候处置。

我们示意图画的那个天审场以前是没有的，它是在江泽民一九九九年七月二十日发动迫害法轮大法后开始兴建的。天审场整个外形是一个巨大的圆柱体，里面就象罗马竞技场那样有圆形的看台。所有参与过迫害法轮大法的人，无论他是什么来头，是国家元首还是什么教派的宗教首领，还是世界首富，或者学界泰斗，如果他在生前没有对迫害法轮功之事弥补过来，这个人死后都绝对上不了天，而且都不由地狱的衙门审理，全部要在天审场这里接受审判，因为现在是创世主弟子修炼时代。

天审场共有两位主审。天审场有一个入口，罪犯从那里被押进来，入口上方看台那个位置上凌空突出一个位置，上面有一个黑色宝座，坐着一位全身散发黑色光芒的神，身上带法轮，来自法轮世界，他是其中一位主审。他的头发是红色的垂直下来，头上是黑色的智慧之光，整个身体被一个圆形的黑色世界包围。在他的身后，是鬼灵市红河范围里的两殿阎王，然后以这位黑色神为中心，看台的半圆范围内都是四大部洲有权柄前来观看的魔的座位，而在这个半圆看台的上空，有层层黑色云透着黑光层层往外扩展，上面都是天上不同层次有权柄前来观看的魔王的座位。

在黑色主审的对面看台那个位置上也有一个同样的白色宝座，上面坐着一位全身散发白色光芒的神，身上也带着法轮，他也来自法轮世界，他是另一位主审。他的头发也是红色的但象西方法官那样卷曲下来，头上是白色的智慧之光，整个身体被一个圆形的白色世界包围。在他的身后，是鬼灵市金河范围的四殿阎王，然后以这位白色神为中心，看台的半圆范围内都是四大部洲有权柄前来观看的神的座位，而在这个半圆看台的上空，有层层白色云透着金光层层往外扩展，上面都是天上不同层次有权柄前来观看的佛道神的座位。

我们看到，从天审场建立到今天，所有这些审判都是全部由这两位来自法轮世界的黑白神主审的。当江某某被押到天审场的时候，前来观看的神和魔太多了，看台上、云上层层都是满满的。

天审场地面的正中央有一个白色的十字，十字外面又有四个红色的圆，圆中央是红色的十字。当二零零二年九月三日这一天从四海中央冥府派出黑白无常执行拘拿江某某元神的时候，无空洞都被清空了，然后黑白无常和众冥府将士押着江某某迅速穿过无空洞直接把江某某押到天审场，在中央白色十字那里跪下，然后黑白无常就退回到入口两边站立，其中白无常一手拿鞭与另一只手交叉在胸前，黑无常一手拿刀与另一只手交叉在胸前。一切就位，天地静默，江某某被锁链捆绑跪在巨大的天审场当中惊恐万分，在那里嚎啕大哭绝望呼喊，懊悔万分。所有后悔在人间都有用，所有后悔来到阴间就再无用处。只见从那四个红色十字交叉的圆那里射出四道光芒击到江某某身上，江某某立即现出蛤蟆原形，绿色皮黑色斑点。

两位主审开始向众神数说江蛤蟆在人世间功过，迫害法轮大法所犯罪行，用了很长时间。如果从鬼灵地狱这里开始，经过地球海中央地狱、然后南海地狱、然后四海中央地狱，这算一遍，那么最后裁定江蛤蟆所犯罪行约需过六十多万遍。裁定完，江蛤蟆就立即被押到地狱入口那里执行刑罚，过一遍后就来到无生之门这里。具体它在地狱里所经过的刑罚我们在后续章节会详细描述。

无生之门前面是一个巨大的深不见底般的石洞，江蛤蟆就被钉在十字架那样的木板上，上面到处都是钉子，然后蛤蟆双手、双脚被锁链捆绑吊在洞里，整个蛤蟆斜着往前倾四十五度角那样。而去年过世的前总理他经过天审场四个红色十字交叉的圆照射之下显露的原形也是人的形象，他所犯罪行被判约需过四万多遍，但因为他在江蛤蟆迫害法轮功前是支持法轮功的，所以他在过一遍地狱刑罚的时候是被厚待的，然后来到无生之门前，双脚站立地上然后双手举起来被洞顶的锁链绑着。

天审场的审判还不是最后的审判，所有参与迫害法轮功的人死后在经过天审场审判后都会被押到无生之门前面那个石洞里关着，等待最后审判之日到来时接受最后的审判。而那些没有参与迫害法轮功被冥府衙门裁定够格入无生之门的经过一遍地狱刑罚后都会被直接投入无生之门直接销毁。我们看到，现在无生之门前面石洞那里等着最后审判的罪犯太多了，我们也数不过来。
